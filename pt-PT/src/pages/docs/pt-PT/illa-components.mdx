### Procurar por um componente

Você pode clicar na aba INSERT no painel direito para entrar na lista de componentes. Digite o nome do componente e procure pelo componente através da caixa de entrada acima.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(34).png'
  alt=""
/>

### Use o manipulador de eventos para acionar eventos

Tomando o componente botão como exemplo, você pode definir um manipulador de eventos para o componente pela forma da imagem mostrada para acionar uma ação ou controlar outros componentes, etc. Quando você clicar em um componente, ou quando os dados do componente mudarem, os eventos correspondentes podem ser acionados.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(35).png'
  alt=""
/>

### Usar os dados do componente em ações

Você pode usar dados de componentes em ações ou outros componentes. Tomando o componente de entrada como exemplo, você pode usar {{input1.value}} para obter o valor inserido no componente. Componentes diferentes podem ter formatos de dados diferentes. Por favor, veja os formatos de dados através do painel esquerdo.

  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(36).png'
  alt=""
