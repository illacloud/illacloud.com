## <Heading hidden>Autenticação de API</Heading>

ILLA pode suportar quase qualquer tipo de autenticação. Nós fizemos parcerias com várias empresas para apoiar todos os tipos padrão de autenticação e também desenvolvemos métodos para suportar quaisquer métodos de autenticação personalizados que não suportamos nativamente.

Você pode usar tokens de autenticação estática ou criar um sistema de autenticação que pede ao usuário final para suas credenciais cada vez que ele acessar o aplicativo.

Esta página fornecerá instruções sobre como autenticar sua API. Se você tiver dúvidas ou precisar de ajuda, você pode entrar em contato com a nossa equipe de suporte usando a bolha Intercom nesta página ou o ícone de ajuda dentro do editor.
