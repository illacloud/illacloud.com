## <Heading hidden>Deploy-Einführung</Heading>

### Wolke

Testen Sie den Ein-Klick-Einsatz in ILLA Cloud für die einfachste Erfahrung.

### Selbstverwalter

Sie können sich lokal mit ILLA CLI, Docker-Komposition, k8s Helm usw. aufhalten. Wir haben den Selbst-Host-Prozess wiederholt so optimiert, dass Sie in wenigen Minuten Selbst-Host- und Software-Implementierungen durchführen können. Im Vergleich zur Cloud können Sie die Produkte von ILLA sicherer und zuverlässiger verwenden und Sie können ILLA Builder verwenden, indem Sie einfach die Bereitstellungsseite nach dem Hosten und Bereitstellen von Builder lokal besuchen.

Weitere Informationen zum Selbstversorger finden Sie auf den ILA CLI, Docker Compose, k8s Steuer-Boot-Seiten.
Wenn Sie Schwierigkeiten mit Ihrer Selbstausführung haben, kontaktieren Sie uns bitte unter GitHub oder business@illasoft.com.

#### Mit ILLA CLI verteilen

Bitte kaufen: [ILLA CLI](https://www.illacloud.com/docs/illa-cli)

#### Manuell verteilen

Oder Sie können ILLA Builder manuell bereitstellen, indem Sie der folgenden Methode folgen:

-   [Docker All-in-One Image](https://github.com/illacloud/deploy-illa-manually/blob/main/docker/README.md)
-   [Docker Compose](https://github.com/illacloud/deploy-illa-manually/blob/main/docker-compose/README.md)
-   [Kubernetes](https://github.com/illacloud/Deploy-illa-manually/blob/main/kubernetes/README.md)
