## <Heading hidden>Вивантажити</Heading>

Завантажити компонент дозволяє користувачам завантажувати файли з локальних машин в будь-яке сховище через API. Далі, ми представляємо, як налаштувати компонент завантаження на нього як зв'язати компонент з хмарним сховищем. як використовувати компонент завантаження для вибору зображення та тощо.

<img src='https://cdn.illacloud.com/official-website/img/docs/upload1.png' alt="" />

### Метод

Ви можете використовувати інші компоненти для управління завантаженням компонентів. Ми підтримуємо наступні три методи:

| очистити перевірку | Використовується для очищення повідомлення підтвердження                             |
| ------------------ | ------------------------------------------------------------------------------------ |
| чисте значення     | Використовується для очищення вибраних                                               |
| значення вимкнено  | Використовується для встановлення компонента завантаження для вимкнених чи включених |

### Обробник подій

Для активації дій при зміні вибраних файлів

### Дані завантажувача

Компонент завантаження містить найбільш часто використовувані дані, які можуть бути викликані в додатку.

`value` - масив з base64 даними файлів. 

`files` - масив об'єктів. Ключі до об'єктів включають `lastModified`, `name`, `size`, `type`. Ви можете використовувати `{{upload1.files.map(file =>file. ame)}}` щоб отримати масив імен файлів, а ви також можете отримати останній змінений масив, розмір масив, введіть масив так само.

#### Завантажити один файл через S3 ресурс

Конфігурація дії виглядає наступним чином. 

Завантажити ім'я об'єкта

```jsx
{{upload1.files.map(файл =>file.name)[0]}}
```

Відправити дані

```jsx
{{upload1.value[0]}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload3.png' alt="" />

#### Завантажити декілька файлів через S3 ресурс

Завантажити список імен об'єкта

```jsx
{{upload1.files.map(файл =>file.name)}}
```

Вивантажити список даних

```jsx
{{upload1.value}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload4.png' alt="" />

Завантажте зображення і відобразити його в компоненті зображення
Змініть джерело зображення на файл base64 дані відвантаження компонента. Дані base64 зберігаються в атрибуті значення у масиві. І ми маємо розбивати префікс файлів даними про файли. Код такий. 

```jsx
{{'data:image/jpeg;base64,'+upload1.value[0]}}
```
