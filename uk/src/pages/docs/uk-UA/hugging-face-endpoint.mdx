## <Heading hidden>Гортування точки дотику</Heading>

З допомогою Hugging Face Inference Endpoints, ви можете легко розгортати Transffusers або будь-яку модель на відданому, повністю керованій інфраструктурі. [Натисніть тут](https://ui.endpoints.huggingface.co/new) щоб створити кінцеву точку. 

### Створити ресурс несправності обличчя

Є два способи додати ресурс Неподружнього Кінцевої точки. 

1.  Введіть закладку "ILLA Builder" >> Натисніть "Ресурси" >> Натисніть "Створити нове" >> Виберіть "Hugging Face Endpoint" >> Налаштування інформації про з'єднання та натисніть "Зберегти ресурс"
2.  Введіть сторінку редагування >> натисніть `+ Новий` в списку дій >> Виберіть "Hugging Face Endpoint" >> Налаштувати інформацію про з'єднання, або натисніть \`+ новий ресурс", щоб додати нову інформацію про підключення

### Налаштувати відомості про підключення

| Властивості  | Обов'язково | Опис                                                                                   |
| ------------ | ----------- | -------------------------------------------------------------------------------------- |
| Ім'я         | обов’язково | Визначте назву ресурсу, яка буде використовуватися для відображення в ILLA             |
| Endpoint URL | обов’язково | створити кінцеву точку тут: https://ui.endpoints.huggingface.co/new і отримати URL.    |
| Маркер       | обов’язково | Маркер організації. Ви можете отримати його в https://huggingface.co/settings/tokens.  |

### Створити дії

Введіть сторінку редагування >> натисніть `+ Нове` в списку дій >> Виберіть "Hugging Face Endpoint\` >> Виберіть існуючий ресурс або додайте новий ресурс

### Налаштування дій

| Властивості   | Обов'язково | Опис                                                                                                                                                                                                                                            |
| ------------- | ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Тип параметра | обов’язково | Тип параметру вашої кінцевої точки. Наприклад, якщо кінцева точка потребує введення тексту, виберіть параметр "input" з текстом. Якщо ваша кінцева точка потребує введення JSON, оберіть заповніть параметр "input" з JSON або значенням ключа. |
| Параметр      | обов’язково | Введіть ваш параметр. Використовувати дані компонентів з {{ componentName.value }}.                                                                                                                                                             |

### Використати регістр

Ми розгорнули \`openai/whisper-base', який отримує дані аудіо-файлу і перетворює їх на текст. Відповідно до зустрічей хвилин, подкастів до смс і т.д. Потім ми представляємо, як використовувати цю модель для створення програми в ILLA Cloud.

#### Крок 1: Побудуйте інтерфейс фронт-енд компонентами

Ми побудували інтерфейс, використовуючи такі компоненти, як завантаження файлів і кнопку, наступним чином.

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf1.png' alt="" />

#### Крок 2: Додати ресурс обличчя Hugging

Заповніть поля, які показані нижче, щоб закінчити конфігурацію ресурсу. [Створіть кінцеву точку](https://ui.endpoints.huggingface.co/new) та отримайте Кінцева точка URL. І отримайте маркер API в [налаштуваннях профілю](https://huggingface.co/settings/tokens). 

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf2.png' alt="" />

#### Крок 3: Налаштувати дію

Спочатку виберіть тип параметра. Взяти `openai/whisper-base` в якості прикладу, для отримання двійкового вхідних файлів файл і змінити параметр на `Binary`.

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf3.png' alt="" />

Встановити бінарний для передавання в модель в якості файлів, завантажених з компоненту завантаження файлу. Наприклад, `{{upload1.value[0]}}`

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf4.png' alt="" />

#### Крок 4: Підключіть компоненти та дії

Додайте обробник подій до кнопки, щоб запустити дію при натисканні на кнопку. Встановіть значення текстового компонента на {{whisper.data[0].text}} для відображення результату конвертації в текстовому компоненті. 

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf5.png' alt="" />

Після налаштування вищезгаданих кроки, додаток створюється і є наступними.

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf6.gif' alt="" />
