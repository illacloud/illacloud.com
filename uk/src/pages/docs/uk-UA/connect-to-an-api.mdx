## <Heading hidden>Підключення до API</Heading>

З'єднатися з будь-яким REST, GraphQL API.

Ви можете під'єднати ILLA до більшості API використовуючи наші нативні інтеграції. Якщо вам потрібно підключитися до внутрішнього API, або якщо не підтримується інтеграція, ви можете підключитися за допомогою нашої інтеграції REST API або GraphQL Integration.

### 1.  Створити API ресурс

Ресурси представляють з'єднання з джерелами даних і є тим, що ви запитуєте в ваших ILLA додатках.Ввійдіть в ILLA і перейдіть в вкладку **Ресурси** > натисніть **Створити новий** та виберіть API, до якого ви хочете підключитися, наприклад, **REST API**. Це відкриває **Налаштувати ресурс** модаль, де ви додаєте інформацію про з'єднання.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

Як мінімум, потрібен базовий URL API, до якого ви хочете підключитися. Ви також можете передати параметри URL, заголовки та ін. Якщо ваш API потребує аутентифікації, ви можете також налаштувати це на цій сторінці. Наприклад, OAuth2.0 вимагає URL-адресу авторизації, Access Token URL тощо. Після введення даних про підключення, натисніть **Ресурс збереження.**

### 2.  Запит вашого API

У редакторі програм, натисніть **+ Нове** в **Списку дій** > вибрати **REST API** і оберіть вже наявного REST API або створіть новий > натисніть **створити дії** та оновити дії з додатковими параметрами URL. заголовки, і т. д.
