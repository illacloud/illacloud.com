<Heading hidden>SQL Generation</Heading>

Мы поддерживаем создание SQL в этих ресурсах: PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB и ClickHouse. Вы можете ввести естественный язык и получить готовый к запуску запрос SQL при использовании этих ресурсов. 

**Введите страницу редактирования приложения и создайте действие, используя один из вышеуказанных ресурсов.** 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate.png' alt="" />

**Выберите тип выписки, который вы хотите генерировать**

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate(1).gif' alt="" />

**Входные данные и нажмите SQL Generate**
