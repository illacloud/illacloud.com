### Перетащите компоненты в список

При перетаскивании компонента списка в Frame, он автоматически включает в себя некоторые компоненты. Вы можете удалить или изменить их и перетащить новые компоненты в список. 

Изменять можно только первый элемент, а другие автоматически будут меняться на основе первого элемента. Вы также можете сосредоточиться на первом элементе и перетащить панель для изменения элементов высоты.

### Отображать результаты запроса в списке

При перетаскивании компонента списка в Frame автоматически отображаются тестовые данные в формате JSON. Вы можете изменить источник данных на действие, используя `{{ mysql_list_query.data }}`. 

После установки источника данных можно изменить данные компонентов:

1.  Выберите компонент в первом пункте. 
2.  Если данные компонента из списка, вы можете задать данные, используя `{{ currentItem.columnName }}` . Например, вы можете установить источник изображения с помощью `{{ currentItem.Avater }}`. `Avater` — это столбец из источника данных списка.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/list2.png'
  alt=""
/>

Вы можете использовать `{{ currentItem.columnName }}` только когда компоненты находятся в списке. 

### Использовать данные списка

Вы можете использовать данные выбранных элементов, используя `{{ list1.selectedItem.columnName }}`. Или используйте данные списка, используя `{{ list1.dataSource[n].columnName }}`
