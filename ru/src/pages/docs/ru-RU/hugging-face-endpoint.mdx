## <Heading hidden>конечная точка лица</Heading>

С помощью точек Endpoint Face Inference вы можете легко развернуть преобразователи, диффузоры или любую модель на выделенной, полностью управляемой инфраструктуре. [Нажмите здесь](https://ui.endpoints.huggingface.co/new), чтобы создать конечную точку. 

### Создать ресурс конечной точки лица

Есть два способа добавить ресурс конечной точки огранки. 

1.  Введите `ILLA Builder` >> Щелкните `Ресурсы` >> Щелкните `Create New` >> Выберите `Hugging Face Endpoint` >> Настройте информацию о соединении и нажмите `Save Resource`
2.  Введите страницу редактирования >> нажмите `+ New` в списке действий >> Выберите `Hugging Face Endpoint` >> Настройте информацию о подключении или нажмите `+ New Resource`, чтобы добавить новую информацию о подключении

### Настройка информации о подключении

| Свойства     | Требуется    | Описание                                                                               |
| ------------ | ------------ | -------------------------------------------------------------------------------------- |
| Наименование | обязательный | Укажите имя ресурса, которое будет использоваться для отображения в ILLA               |
| Endpoint URL | обязательный | создать конечную точку здесь: https://ui.endpoints.huggingface.co/new и получить URL.  |
| Токен        | обязательный | Маркер организации. Вы можете получить его в https://huggingface.co/settings/tokens.   |

### Создать действия

Введите страницу редактирования >> нажмите `+ New` в списке действий >> Выберите `Hugging Face Endpoint` >> Выберите существующий ресурс или добавьте новый ресурс

### Настройка действий

| Свойства      | Требуется    | Описание                                                                                                                                                                                                                                                          |
| ------------- | ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Тип параметра | обязательный | Тип параметра конечной точки. Например, если ваша конечная точка нуждается в вводе текста, выберите заполните параметр «inputs» с текстом. Если ваша конечная точка нуждается в JSON, выберите заполните параметр «inputs» с помощью JSON или ключевого значения. |
| Параметр      | обязательный | Введите ваш параметр. Используйте {{ componentName.value }} , чтобы использовать данные компонентов.                                                                                                                                                              |

### Вариант использования

Мы развернули `openai/whisper-base`, который получает ввод аудио файла и конвертирует в текст. Он подходит для встреч минут, подкастов к тексту и т.д. Далее мы покажем, как использовать эту модель для построения приложения в облаке ILLA.

#### Шаг 1: Постройте внешний интерфейс с компонентами

Мы создали интерфейс, используя компоненты, такие как загрузка файлов и кнопка, следующим образом:

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf1.png' alt="" />

#### Шаг 2: Добавьте ресурс Hugging Face

Заполните поля ниже для завершения настройки ресурса. [Создайте конечную точку](https://ui.endpoints.huggingface.co/new) и получите URL Endpoint URL. И получите API токен организации в [настройках профиля](https://huggingface.co/settings/tokens). 

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf2.png' alt="" />

#### Шаг 3: Настройка действия

Сначала выберите тип параметра. Возьмите в качестве примера `openai/whisper-base` эту модель требует ввода бинарного файла, поэтому мы изменим параметр на `Binary`.

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf3.png' alt="" />

Установить бинарный файл, который будет передан модели в качестве файловых данных, загруженных из компонента загрузки файла. Например, `{{upload1.value[0]}}`

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf4.png' alt="" />

#### Шаг 4: Соедините компоненты и действия

Добавьте обработчик событий к кнопке, чтобы запустить действие, когда кнопка нажата. И установите значение текстового компонента в {{whisper.data[0].text}} , чтобы отобразить результат преобразования текстового компонента. 

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf5.png' alt="" />
