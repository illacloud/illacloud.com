## <Heading hidden>трансформатор</Heading>

Мы поставили трансформатор в ILLA для удовлетворения ваших требований для написания JavaScript для уточнения логики вашего приложения. В преобразователе вам не нужно писать код в JavaScript с помощью {{template syntax}} , как вы делаете в поле ввода символов, так что вы можете свободно писать длинные и сложные блоки JS кода. Кроме того, Transform помогает разработчикам, ответственным за дизайн страниц и построение логики, более эффективно выполнять свои работы по разработке, не ограничиваясь прогрессом друг друга.

Трансформер можно использовать не только для обычных операций с данными, но и для создания операций преобразования данных. Было бы лучше выбрать независимый трансформатор, когда у вас есть несколько операций для преобразования данных, или необходимо использовать сложную логику JavaScript для преобразования данных.

### Творческие преобразователи

Щелкните на вкладке Преобразователи в списке действий и нажмите на кнопку "+ New" для создания трансформатора. Он будет отображаться в списке действий после создания. Вы можете изменить имя, написать код JS, сохранить и запустить его в панели настроек после выбора.

> Вы можете использовать кнопку «Просмотр», чтобы увидеть результаты трансформатора. Обратите внимание, что для получения нового значения в остальной части приложения необходимо сохранить трансформатор.

Вставка вызывает трансформатор со значением, равным данным, возвращаемым внутри трансформатора, эти данные будут изменены по мере изменения данных, введенных в преобразователь с кодом преобразователя. Так же, как и в поле ввода символов, вы можете использовать синтаксис шаблона для вызова глобальной переменной No. Когда вы пишете код JS для вызова данных, вы можете найти конкретное имя переменной данных в рабочей области данных в левой ручной панели, чтобы помочь вам быстрее писать код.

Вы можете вызвать переменную трансформатора в поле ввода данных, используя `{{transformer name. tag name}}` для вызова переменной трансформатора в поле ввода данных.

### Трансформатор запроса

Мы поместили трансформатор в операцию запроса, чтобы позволить вам преобразовать загруженные данные в желаемый формат сразу после завершения действия Вы можете просто передать {{actionName.data}} для получения данных после завершения преобразования в трансформаторе в рамках действия.
