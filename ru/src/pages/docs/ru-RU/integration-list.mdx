## <Heading hidden>Список Интеграций</Heading>

ILLA предлагает ряд встроенных интеграций, позволяющих легко подключаться к источникам данных с минимальной конфигурацией. Просто выберите соответствующую интеграцию из списка доступных опций, введите необходимые учетные данные, и вы можете начать использовать источник данных в ILLA. Это устраняет необходимость в настройке сложных конфигураций и позволяет быстро начать работу с вашими данными.

### Интеграция с базой данных

| Наименование     | Описание                                                                                                                                                                                                                                           |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql            | MySQL - это система управления реляционными базами данных с открытым исходным кодом (RDBMS), основанная на структурированном языке запросов (SQL).                                                                                                 |
| PostgreSQL       | PostgreSQL является с открытым исходным кодом объектно-реляционной системой управления базами данных (ORDBMS), известной своей надежностью и богатством возможностей.                                                                              |
| MariaDB          | MariaDB - это реляционная система управления базами данных с открытым исходным кодом, которая является форком базы данных MySQL.                                                                                                                   |
| TiDB             | TiDB - это база данных NewSQL, поддерживающая загрузку Hybrid Transactional and Analytical Processing.                                                                                                                                             |
| Редис            | Redis это хранилище данных с открытым исходным кодом, которое часто используется в качестве базы данных, кэша и брокера сообщений.                                                                                                                 |
| MongoDB          | MongoDB - это бесплатная программа для кросс-платформенных документоориентированных баз данных.                                                                                                                                                    |
| Эластичный поиск | Elasticsearch - это открытый и распределенный поисковый и аналитический механизм, построенный поверх Apache Lucene.                                                                                                                                |
| Firebase         | Firebase - это облачная платформа для разработки мобильных приложений и веб-приложений, разработанная компанией Google.                                                                                                                            |
| Supabase DB      | Supabase - это платформа с открытым исходным кодом для создания приложений без сервера в реальном времени. Она построена поверх PostgreSQL и предоставляет ряд инструментов и услуг для быстрого создания и развертывания безсерверных приложений. |
| Clickhouse       | ClickHouse - это система управления базами данных с открытым исходным кодом, которая предназначена для интерактивной аналитической обработки данных (OLAP) и анализа данных в режиме реального времени.                                            |

### Интеграции с API

| Наименование         | Описание                                                                                                                                        |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| API REST             | REST – это архитектурный стиль для построения веб-сервисов. RESTful API - это API, которые следуют архитектурному стилю REST.                   |
| Amazon S3 (Общий S3) | Amazon S3 - это сервис хранения объектов, предлагаемый AWS. Он обеспечивает безопасное, масштабируемое и недорогое хранение данных в Интернете. |
| SMTP                 | SMTP (Simple Mail Transfer Protocol) - это протокол для отправки и получения сообщений по интернету.                                            |
| GraphQL              | GraphQL является языком запросов для API и временем выполнения этих запросов с вашими существующими данными.                                    |
