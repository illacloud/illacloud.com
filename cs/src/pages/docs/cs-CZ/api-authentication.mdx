## <Heading hidden>API ověření</Heading>

ILLA může podporovat téměř jakýkoli druh ověřování. Spolupracujeme s různými společnostmi, abychom podpořili všechny standardní typy autentizace, a také jsme vyvinuli metody na podporu jakýchkoliv vlastních autentizačních metod, které doma nepodporujeme.

Můžete použít statické ověřovací tokeny nebo vytvořit ověřovací systém, který bude konečného uživatele volat po každém přístupu k aplikaci.

Tato stránka vám poskytne pokyny, jak ověřit váš API. Pokud máte dotazy nebo potřebujete pomoc, můžete kontaktovat náš tým podpory pomocí bubliny Intercom na této stránce nebo ikony nápovědy uvnitř editoru.
