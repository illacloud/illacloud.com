## <Heading hidden>Integreringslista</Heading>

ILLA erbjuder en rad färdiga integrationer som gör att du enkelt kan ansluta till dina datakällor med minimal konfiguration. Välj helt enkelt lämplig integration från listan över tillgängliga alternativ, ange nödvändiga uppgifter, och du kan börja använda din datakälla inom ILLA. Detta eliminerar behovet av att konfigurera komplexa konfigurationer och gör det möjligt för dig att snabbt börja arbeta med dina data.

### Databasintegrationer

| Namn             | Beskrivning                                                                                                                                                                                                                            |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql            | MySQL är ett relationsbaserat databashanteringssystem med öppen källkod (RDBMS) som är baserat på det strukturerade frågespråket (SQL).                                                                                                |
| PostgreSQL       | PostgreSQL är en öppen källkod objekt-relationellt databashanteringssystem (ORDBMS) som är känd för sin tillförlitlighet och funktions-rikedom.                                                                                        |
| MariaDB          | MariaDB är ett relationellt databashanteringssystem med öppen källkod som är en gaffel för MySQL-databasen.                                                                                                                            |
| TiDB             | TiDB är en öppen källkod NewSQL-databas som stöder Hybrid Transactional och Analytical Processing arbetsbelastning.                                                                                                                    |
| Redis            | Redis är en öppen källkod i minnet datastruktur butik som ofta används som en databas, cache, och meddelande mäklare.                                                                                                                  |
| MongoDB          | MongoDB är en fri och öppen källkod plattformsoberoende dokument-orienterade databasprogram.                                                                                                                                           |
| Elastisk sökning | Elasticsearch är en öppen källkod, distribuerad sök-och analytics motor som bygger på toppen av Apache Lucene.                                                                                                                         |
| Firebase         | Firebase är en molnbaserad mobil- och webbapplikationsutvecklingsplattform som utvecklats av Google.                                                                                                                                   |
| Supabase DB      | Supabase är en plattform med öppen källkod för att bygga serverlösa applikationer i realtid. Det är byggt på toppen av PostgreSQL och erbjuder ett antal verktyg och tjänster för att snabbt bygga och distribuera serverlösa program. |
| Clickhouse       | ClickHouse är ett kolumnorienterat databashanteringssystem med öppen källkod som är utformat för onlineanalys (OLAP) och dataanalys i realtid.                                                                                         |

### API integrationer

| Namn                  | Beskrivning                                                                                                                   |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| REST API              | REST är en arkitektonisk stil för att bygga webbtjänster. RESTful API:er är API:er som följer den REST arkitektoniska stilen. |
| Amazon S3(General S3) | Amazon S3 är en objektlagringstjänst som erbjuds av AWS. Det ger säker, skalbar och billig lagring för data på internet.      |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) är ett protokoll för att skicka och ta emot e-postmeddelanden via internet.              |
| GraphQL               | GraphQL är ett frågespråk för API:er och en körtid för att uppfylla dessa frågor med dina befintliga data.                    |
