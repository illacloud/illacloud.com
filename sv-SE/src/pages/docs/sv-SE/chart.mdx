## <Heading hidden>Diagram</Heading>

### Visa data i ett diagram

Du kan använda åtgärder och transformatorer för att bygga diagram. Du kan referera till en array eller ett objekt i fältet **Datakälla** för ett diagram. 

När du väljer en datakälla, analyserar ILLA data och fyller diagrammet. Som standard fyller den första icke-numeriska kolumnen **X-axeln** men du kan ändra detta och andra egenskaper som **Sjökortstyp**, **Gruppera av**, etc.
