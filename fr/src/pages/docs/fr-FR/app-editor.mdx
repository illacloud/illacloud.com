## <Heading hidden>Éditeur d'application</Heading>

L'éditeur APP est l'interface pour développer l'application et se compose des parties principales suivantes :

-   Barre d'outils
-   Frame
-   Éditeur d'action
-   Espace de travail des données
-   Insert
-   Inspecter

<img alt="editor" src={'https://cdn.illacloud.com/official-website/img/official-site/content.png'} />

### Barre d'outils

La barre d'outils est située en haut de l'éditeur APP qui est principalement utilisé pour configurer l'APP et peut être utilisé pour les tâches suivantes：

-   Cliquez sur l’icône ILLA à gauche pour revenir à la page d’accueil.
-   Voir les informations de l'application et enregistrer le statut.
-   Afficher/masquer les éléments à gauche, à droite, le panneau inférieur.
-   Ajustement de la mise à l'échelle des éléments.
-   Afficher tous les développeurs actuels de l'APP.
-   Alerte l'application à l'état actuel des bogues.
-   Partagez l'application avec d'autres développeurs.
-   Exécution de l'APP.

#### Plus de fonctionnalités sont à venir

-   Importer des APP depuis json.
-   Reproduire les APPs.
-   Exporter les APPs au format json.
-   Sélectionnez la version souhaitée de l'APP.
-   Voir l'historique modifié.
-   Recharger toutes les actions.

### Frame

Le cadre est la zone de l'APP où vous pouvez modifier l'interface graphique de l'APP. Vous pouvez glisser et déposer des inserts dans le cadre à partir du panneau des inserts sur la main droite ; vous pouvez également sélectionner les inserts qui ont été placés dans le cadre et utiliser la souris pour glisser-déposer pour modifier la position et la taille des inserts.

Lorsque vous faites glisser et déposer des inserts ou que vous modifiez leur taille, un carré pointillé apparaîtra en arrière-plan pour vous aider à organiser les insertions dans le cadre. Lorsque vous rencontrez des chevauchements d'insertions pendant le glisser-déposer, ILLA Builder donnera la priorité à l'insertion que vous déplacez et déplacera automatiquement le bloc hors de la voie pour vous.

#### Aperçu

Le bouton d'aperçu se trouve dans le coin supérieur droit du cadre . Lorsqu'il est cliqué, l'éditeur passera en mode prévisualisation, les panneaux en bas à gauche et à droite seront cachés et les inserts dans le cadre ne seront pas modifiés, toutes les inserts seront présentés devant vous en taille 1:1.

> ### Navigateurs pris en charge
>
> Nous vous recommandons d'ouvrir ILLA Builder dans Google Chrome.

### Éditeur d'action

L'éditeur d'action est situé sous le cadre et est la zone où vous écrivez du code pour interagir avec les données des ressources. Dans cette zone, vous pouvez trouver des opérations créées, sélectionner la ressource opérationnelle courante, écrire des requêtes et configurer des transformateurs. ou peut modifier l'instruction de requête dans l'éditeur d'action et lorsque vous mettez à jour l'instruction de requête, l'éditeur APP mettra à jour automatiquement vos résultats d'action.

Lorsque vous entrez des instructions d'action SQL, vous pouvez choisir d'utiliser le mode SQL pour les éditer via un code simple, ou vous pouvez choisir le mode graphique pour les modifier sous le cadre d'action que nous vous donnons pour compléter rapidement l'édition de l'action. ou des tutoriels détaillés, veuillez consulter la documentation d'exploitation.

### Liste des actions

La liste des actions se trouve sur le côté gauche du panneau de configuration, où vous pouvez trouver l'action dont vous avez besoin pour l'utiliser et la modifier dans l'éditeur d'édition. ou peut également ajouter une nouvelle action ici, voir les détails du processus de création.

### Espace de travail des données

L'espace de travail des données est situé dans le panneau à gauche du cadre, où vous pouvez voir les insertions, et le statut global de l'éditeur, tous affichés au format Json. Vous pouvez également modifier les données en les sélectionnant dans l'espace de travail ou en les supprimant en cliquant sur le bouton Paramètres supplémentaires.

### Insert

Le Hors-texte se trouve dans le panneau à droite du cadre, et vous pouvez déplacer l'insertion vers le cadre en le glissant et en le déposant avec la souris. Vous pouvez trouver l'insertion dont vous avez besoin par type d'insertion, ou vous pouvez la rechercher et la trouver rapidement.

### Inspecter

L'inspection se trouve dans le panneau à droite du cadre. Lorsque vous sélectionnez une insertion dans l'espace de travail cadre ou de données, le panneau de droite bascule automatiquement sur la page d'insertion. À partir de cette page, vous pouvez définir des variables, mettre à jour le nom d'insertion, créer des gestionnaires d'événements ou effectuer une opération de suppression d'insertion. La barre d'outils de l'insertion sera également réglée différemment pour différents inserts, de sorte que vous pouvez modifier le contenu des différents inserts.
