## <Heading hidden>Déployer l'introduction</Heading>

### Nuage

Essayez un déploiement en un clic dans le Cloud ILLA pour une expérience plus facile.

### Auto-hôte

Vous pouvez vous héberger localement avec ILLA CLI, Docker compose, k8s helm etc. Nous avons à plusieurs reprises optimisé le processus d'auto-hôte pour vous permettre de faire des déploiements de logiciels et d'hébergeurs en quelques minutes. Par rapport au nuage, vous pouvez utiliser les produits ILLA de manière plus sûre et plus fiable, et vous pouvez utiliser ILLA Builder en visitant simplement le site de déploiement après avoir hébergé et déployé le Builder localement.

Pour plus d'informations sur l'auto-hôte, veuillez consulter la page de démarrage de l'ILA CLI, Docker compose, k8s helm boot pages.
Si vous avez des difficultés avec votre autodéploiement, veuillez nous contacter sur GitHub ou business@illasoft.com.

#### Déployer avec ILLA CLI

Veuillez vérifier : [ILLA CLI](https://www.illacloud.com/docs/illa-cli)
