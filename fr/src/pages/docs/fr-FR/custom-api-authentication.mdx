## <Heading hidden>Authentification API personnalisée</Heading>

ILLA supporte par défaut de nombreuses normes d'authentification (voir ici pour plus d'informations). 
De plus, ILLA possède un mode d'authentification personnalisée qui vous permet de combiner plusieurs étapes en prenant en charge pratiquement n'importe quelle authentification API. 
Ces méthodes d'authentification peuvent être utilisées pour les ressources REST et GraphQL.
