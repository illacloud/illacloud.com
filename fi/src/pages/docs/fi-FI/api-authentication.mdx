## <Heading hidden>API todennus</Heading>

ILLA voi tukea lähes kaikenlaista todentamista. Olemme tehneet yhteistyötä eri yritysten kanssa tukeaksemme kaikkia vakiomuotoisia autentikointeja ja olemme myös kehittäneet menetelmiä tukemaan kaikkia mukautettuja autentikointimenetelmiä, joita emme natiivisti kannata.

Voit käyttää staattista tunnistautumispoletteja tai luoda tunnistautumisjärjestelmän, joka pyytää loppukäyttäjää käyttäjätunnuksistaan aina kun he käyttävät sovellusta.

Tämä sivu antaa sinulle ohjeet siitä, miten API todennetaan. Jos sinulla on kysymyksiä tai tarvitset apua, voit ottaa yhteyttä tukitiimiin käyttämällä tällä sivulla olevaa Intercom-kuplaa tai editorin sisällä olevaa ohjekuvaketta.
