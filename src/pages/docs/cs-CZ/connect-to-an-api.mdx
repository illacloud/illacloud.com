## <Heading hidden>Připojte se k API</Heading>

Připojte se k jakémukoliv REST, GraphQL API.

Můžete připojit ILLA k většině API pomocí našich původních integrací. Pokud se musíte připojit k internímu API, nebo pokud neexistuje podporovaná integrace, se můžete připojit pomocí naší integrace REST API nebo integrace s GraphQL.

### 1.  Vytvořit zdroj API

Zdroje představují připojení ke zdrojům dat a jsou tím, co dotazujete ve vašich aplikacích ILLA.Přihlaste se do ILLA a přejděte na záložku **zdroje** > klikněte **Vytvořit nové** a vyberte API, ke kterému se chcete připojit. např. **RESTRET API**. Tímto se otevře **Konfigurovat zdrojový** modál, kde přidáte informace o připojení.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

Minimálně potřebujete základní URL API, ke které se chcete připojit. Můžete také zadat URL parametry, záhlaví, atd. Pokud vaše API vyžaduje autentizaci, můžete to také konfigurovat na této stránce. Například OAuth2.0 vyžaduje autorizační URL, URL přístupového tokenu atd. Po zadání údajů o připojení klikněte na **Uložit zdroje.**

### 2.  Dotaz na vaše API

V editoru aplikací, klikněte **+ Nový** v **seznamu akcí** > vyberte **REZT API** a vyberte existující REZT API nebo vytvořte nové> klikněte **vytvořit akci** a aktualizujte akce o dalších parametrech URL, záhlaví atd.
