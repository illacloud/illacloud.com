## <Heading hidden>Radio-Button</Heading>

Ein Funkknopf wählt die Eingabekomponente aus, mit der Benutzer eine Option aus einer Gruppe vordefinierter Optionen auswählen können. Es erscheint als abgerundete quadratische Schaltfläche, die von den Benutzern ausgewählt oder abgewählt werden kann. Wenn ein Auswahlknopf ausgewählt ist, werden alle anderen Radioknöpfe in der gleichen Gruppe automatisch deaktiviert, stellen Sie sicher, dass nur eine Option gleichzeitig ausgewählt werden kann. Funkknöpfe werden häufig in Formularen, Fragebögen und Umfragen verwendet, um Benutzern zu helfen, eine einzige Auswahl aus einer Liste von Optionen zu treffen.

### Eigenschaften

#### Manuelle Optionen

Fügen Sie neue Optionen hinzu oder löschen Sie die Optionen manuell und konfigurieren Sie sie einzeln. Die Eigenschaften der Optionen sind: 

| Eigenschaften | Beschreibung                                  |
| ------------- | --------------------------------------------- |
| Label         | Der angezeigte Text für die Option            |
| Wert          | Der der Option zugeordnete Wert               |
| Deaktiviert   | Ob die Option für die Auswahl deaktiviert ist |

#### Zugeordnete Optionen

Sie können auch Optionen aus einer Datenquelle hinzufügen, indem Sie die Label- und Werteeigenschaften den entsprechenden Datenfeldern zuordnen. Auf diese Weise können Sie Optionen dynamisch aus der Datenbank beziehen. Am Ende dieses Artikels, werden wir zeigen, wie man es verwenden durch ein Beispiel.

| Eigenschaften | Beschreibung                                                                           |
| ------------- | -------------------------------------------------------------------------------------- |
| Datenquelle   | Datenquelle für Optionen festlegen                                                     |
| Label         | Verwenden Sie {{item}} , um die Bezeichnung der Optionen zu setzen                     |
| Wert          | Verwenden Sie {{item}} , um den Wert der Optionen zu setzen                            |
| Deaktiviert   | Verwenden Sie {{item}} , um festzulegen, ob die Option für die Auswahl deaktiviert ist |

### Methode

Sie können andere Komponenten verwenden, um die Komponente zu steuern. Wir unterstützen die folgenden drei Methoden:

#### setWert

Um die ausgewählte Option zu setzen, zum Beispiel {{”value1”}}

| Eigenschaften | Beschreibung                              |
| ------------- | ----------------------------------------- |
| Wert          | Ein String mit ausgewähltem Optionswert.  |

#### clearWert

Ausgewählte Optionen löschen

#### validate

Um zu überprüfen, ob die Eingabe zulässig ist

#### löschen

Um die Validierungsmeldung zu löschen

### Event-Handler

Wir unterstützen das Abhören der "onChange" Veranstaltung der Funk-Button-Komponente. 

Wenn sich die ausgewählte Option ändert, kann sie eine bestimmte Aktion auslösen. Das bedeutet, wenn ein Benutzer eine andere Option als eine Gruppe von Radio-Buttons wählt, eine bestimmte Aktion kann basierend auf der gewählten Option durchgeführt werden. Zum Beispiel in einem Webformular die Auswahl einer anderen Option kann die Anzeige von verschiedenen Formularfeldern oder die Übermittlung des Formulars an einen anderen Bestimmungsort auslösen. Auf diese Weise kann sich die vom Benutzer gewählte Option auf das Verhalten der von ihm verwendeten Software oder Web-Anwendung auswirken. Die Möglichkeit, spezielle Aktionen auf Basis von Benutzereingaben auszulösen, ist ein wichtiger Aspekt des interaktiven Software-Designs und kann die Benutzerfreundlichkeit und Funktionalität der Software verbessern.

### Gehäuse verwenden

Als nächstes zeigen wir, wie man den zugeordneten Modus verwendet, um dynamisch Optionen zu erhalten. 

#### Schritt 1 Eine Aktion hinzufügen

Wir haben eine Tabelle namens `selection` mit 5 Spalten erstellt: `options_de`, `options_jp`, `options_zh`, `options_kr`, `value`. Wir haben die Optionswerte in `value` gespeichert und in verschiedenen Sprachen in anderen Spalten gespeichert. 

Erstelle eine Aktion um alle Daten in `selection` mit dem Namen `postgresql1` aufzulisten

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/radio1.png' alt="" />

```jsx
* aus der Auswahl auswählen
```

#### Schritt 2 die Komponente konfigurieren

1.  Datenquellen auf `{{postgresql1.data}} ` gesetzt 
2.  Konfigurieren Sie das Label, um das Label basierend auf der Sprache zu ändern. 

    1.  Verwenden Sie {{ item.columnName }} , um die Spalte festzulegen. 
    2.  Verwenden Sie {{ currentUserInfo.language }} , um die Sprache der Benutzer in der ILLA festzulegen.

    ```jsx
    {{currentUserInfo.language=='ja-JP' ? item.options_jp 
    : currentUserInfo.language == 'ko-KR' ? item.options_kr 
    : currentUserInfo.language == 'zh-CN' ? item.options_zh 
    : item.options_de }}
    ```
