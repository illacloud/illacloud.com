## <Heading hidden>Benutzerdefinierte API-Authentifizierung</Heading>

ILLA unterstützt standardmäßig viele Authentifizierungsstandards (siehe hier für weitere Informationen). 
Zusätzlich verfügt ILLA über einen benutzerdefinierten Authentifizierungsmodus, mit dem Sie mehrere Schritte kombinieren können, die praktisch jede API-Authentifizierung unterstützen. 
Diese Authentifizierungsmethoden können sowohl für REST als auch für GraphQL Ressourcen verwendet werden.
