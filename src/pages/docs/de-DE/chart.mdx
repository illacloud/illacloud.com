## <Heading hidden>Diagramm</Heading>

### Daten in einem Diagramm anzeigen

Sie können Aktionen und Transformatoren verwenden, um Diagramme zu erstellen. Sie können ein Array oder ein Objekt im Feld **Datenquelle** für ein Diagramm referenzieren. 

Nachdem Sie eine Datenquelle ausgewählt haben, analysiert ILLA die Daten und füllt das Diagramm aus. Standardmäßig wird die erste nicht-numerische Spalte die **X-Achse** gefüllt, aber Sie können diese und andere Eigenschaften wie den **Diagramm-Typ**, **Gruppe by**, etc. ändern.
