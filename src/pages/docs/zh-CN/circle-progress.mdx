将输入组件的占位符设置为“请将您的名字放在这里”，以及下面显示的数字输入的默认值。

#### 步骤 2 配置组件

对于**`input`** 和 **`number input`** 组件, 我们可以配置它们来将其设置为 **`circle progress`** 组件, 以反映完成的进度。

1.  点击 **`input`** 和 **`number input`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更新的 **\`circle progress** 组件作为事件的目标。
3.  选择 **`setValue` ** 动作并将其设置为

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![cir_event_config](https://cdn.illacloud.com/official-website/img/official-site/components/cir_event_config.jpg)

注意：需要设置上述事件处理程序到 输入和数字输入组件。

关于“提交”**`按钮`** 组件，如果表单已完成，我们可以配置它来提交表单(当circle progress 100%)

1.  点击 **`按钮`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择 **`form`** 你想要更新的组件作为事件的目标。
3.  选择 **\`提交** 方法，并将 `{{{circleProgress1.value == 100}` 放入**`仅在`** 时运行

![cir_sub_config](https://cdn.illacloud.com/official-website/img/official-site/components/cir_sub_config.jpg)
