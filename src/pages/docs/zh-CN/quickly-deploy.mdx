## <Heading hidden>快速部署</Heading>

您可以非常快速地部署ILLA，请遵循下面的指令：

首先，请下载这个[docker file]\(https://github.com/illacloud/display-illa-manual y/tree/main/docker)到您的计算机，然后运行以下代码。

```jsx
cd docker;
/bin/bash ./scripts/run-official-image.sh;
```

使用默认用户名和密码登录：

用户名: **`root`**

密码: **`password`** (仅自主模式)。
