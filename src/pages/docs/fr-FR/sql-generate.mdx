<Heading hidden>Génération SQL</Heading>

Nous supportons la génération de SQL dans ces ressources : PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB et ClickHouse. Vous pouvez entrer le langage naturel et obtenir une requête SQL prête à être exécutée en utilisant ces ressources. 

**Entrez la page d'édition de l'application et créez une action en utilisant l'une des ressources ci-dessus.** 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate.png' alt="" />

**Sélectionnez le type d'instruction que vous voulez générer**

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate(1).gif' alt="" />

**Renseignez les détails et cliquez sur SQL Generate**

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate(2).png' alt="" />
