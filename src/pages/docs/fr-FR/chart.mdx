## <Heading hidden>Graphique</Heading>

### Afficher les données dans un graphique

Vous pouvez utiliser des actions et des transformateurs pour construire des graphiques. Vous pouvez référencer un tableau ou un objet dans le champ **Data source** pour un graphique. 

Après avoir sélectionné une source de données, ILLA analyse les données et remplit le graphique. Par défaut, la première colonne non-numérique remplit l'axe des **X-axis** mais vous pouvez changer cette propriété et d'autres propriétés comme **Type de graphique**, **Grouper par**, etc.
