## <Heading hidden>Authentification API</Heading>

L'ILLA peut supporter presque tous les types d'authentification. Nous avons établi des partenariats avec différentes entreprises pour supporter tous les types d'authentification standard et avons également développé des méthodes pour supporter toutes les méthodes d'authentification personnalisées que nous ne supportons pas nativement.

Vous pouvez utiliser des jetons d'authentification statiques ou créer un système d'authentification qui invite l'utilisateur final à récupérer ses identifiants chaque fois qu'il accède à l'application.

Cette page vous fournira des instructions sur la façon d'authentifier votre API. Si vous avez des questions ou besoin d'aide, vous pouvez contacter notre équipe de support en utilisant la bulle Intercom sur cette page ou l'icône d'aide à l'intérieur de l'éditeur.
