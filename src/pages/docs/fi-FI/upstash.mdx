## <h2 hidden>Mitä yläkerta on?</h2>

Upstash on pilvi-natiivi palvelinmaton palvelu, jossa Redis on yhteensopiva joka tarjoaa helpon tavan integroida Redis toiminnallisuus sovelluksiin. ILLA Cloudin yhteydessä Upstash -integraatio antaa käyttäjille mahdollisuuden hyödyntää Upstashin ominaisuuksia ja ominaisuuksia ILLA Cloud -sovelluksessa.

Tässä tutoriaalissa hahmotellaan prosessi, jossa luodaan Admin Panel ILLA Builderia ja Upstashia käyttäen muutamassa yksinkertaisessa vaiheessa. ILLA Cloud on matalan koodin alusta kehittäjille, joka mahdollistaa sisäisten työkalujen nopean kehittämisen ja käyttöönoton. Se mahdollistaa sivujen luomisen vetämällä ja pudottamalla käyttöliittymän komponentteja, yhdistämällä mihin tahansa tietokantaan tai APIin ja kirjoittamalla JavaScriptiä. Jos haluat lisätietoja Upstashista, vieraile heidän verkkosivustollaan osoitteessa https://upstash.com/(https://upstash.com/) Aloitetaan!

### Määritä Takaisin loppu ylhäällä

Luo tili yläkerrassa. Voit kirjautua sisään käyttämällä Amazon, Github tai Google-tilejä vaihtoehtoisesti.

Kun olet kirjautunut tilillesi `Upstash`, klikkaa `Create database`-painiketta rullalla.

![up_home](https://cdn.illacloud.com/official-website/img/docs/up_home.jpg)

Paina `Upstash`-kojelautaa `Luo tietokanta`.

![up_create_db](https://cdn.illacloud.com/official-website/img/docs/up_create_db.jpg)

Huomautus: sinulla voi olla vain yksi tietokanta vapaassa tasossa.

Tietokannan kokoonpanoa varten voit nimetä tietokannan niin kuin haluat. Täällä me nimitämme sen `db1`. Valitse `primary region` ja `read region` kirjoittaaksesi ja lukeaksesi käyttäjän sijainteja. Ota Käyttöön `TSL/SSL`. Se on valinnainen ottaa käyttöön Eviction.

![up_db_config](https://cdn.illacloud.com/official-website/img/docs/up_db_config.jpg)

Napsauta **`create`** -painiketta, meillä on nyt käytössä oleva tietokanta.

### Luo yläkerta ILLA Cloudissa

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu sisään Illa tilillesi, valitse `**Resurssit **` sivun yläreunassa ja napsauta `**Create New**` painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Upstash`** tietokantaluettelosta.

![action_list_0](https://cdn.illacloud.com/official-website/img/docs/action_list_0.jpg)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Resurssin luomisen jälkeen valmis `Upstash` näkyy kuten on esitetty.

![up_ex_ready](https://cdn.illacloud.com/official-website/img/docs/up_ex_ready.jpg)

-   Luo rakentajaan

Kirjaudu sisään Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`Upstash`** tietokantaluettelosta. Yhdistä tietokantaan sitten alla olevassa kohdassa `Connection Settings` kuvatut parametrit.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Tältä osin meidän on annettava tietoa Upstash-tietokantaan yhdistämistä varten.

![up_config](https://cdn.illacloud.com/official-website/img/docs/up_config_rev.jpg)

| Ominaisuudet   | Pakollinen | Kuvaus                                                                                                                 |
| -------------- | ---------- | ---------------------------------------------------------------------------------------------------------------------- |
| Nimi           | vaadittu   | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                            |
| Hostname       | vaadittu   | Tietokannan URL-osoite tai IP-osoite                                                                                   |
| Portti         | vaadittu   | Palvelimen portti numero, jota sinun pitäisi käyttää yhdistämiseen. Jos et määritä porttia, oletusportti on '6379'.    |
| Salasana       | vaadittu   | Käytä tätä salasanaa tunnistautumiseen.                                                                                |
| SSL vaihtoehto | vaadittu   | ottaako SSL käyttöön (suositeltavaa ottaa käyttöön)                                                                    |

Jotta voidaan täyttää vaaditut tiedot, mene tietokannan kojelauta sivu ja klikkaa tietokantaa, jonka olemme juuri luoneet ylästash.

![up_info](https://cdn.illacloud.com/official-website/img/docs/up_info.jpg)

Voit kopioida sen leimaamalla hiiren päätepisteen, salasanan ja portin päälle ja klikkaamalla sen takana olevaa kopiointitekstikuvaketta.

Upstash integraation konfiguraatio ILLA Cloud, laita nimi haluat `Nimi`, `Endpoint` hakemistoon `hostname`, annettu `Port` hakemistoon `Port`, ja kopioi salattu `Password`.

![up_config_done](https://cdn.illacloud.com/official-website/img/docs/up_config_rev_done.jpg)

### Määritä Ylhäällä

| Ominaisuudet  | Kuvaus                                                                                 |
| ------------- | -------------------------------------------------------------------------------------- |
| Redis kysely  | käytä kyselyä Redis-tietokantaan tallennettujen tietojen hakemiseen ja käsittelemiseen |
| Muuntaja      | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                             |

Esimerkin käyttö:

1.  Hanki Arvot:
    -   Hanki arvo: Käytä **`GET`** -komentoa noutaaksesi tiettyyn avaimeen liittyvän arvon. Esimerkiksi: **`GET avain`**.
2.  Aseta Arvot:
    -   Aseta arvo: Käytä **`SET`** -komentoa asettaaksesi avaimen arvon. Esimerkiksi **`SET avain arvo`**.
3.  Hash-toiminnot:
    -   Aseta kenttäarvon parit hashissa: Käytä komentoa **`HSET`** asettaaksesi kenttäarvon parin hash. Esimerkiksi: **`HSET hashKey kenttäarvo`**.
    -   Hanki arvo hashista: Käytä **`HGET`** -komentoa noutaaksesi tietyn kentän arvon hash-tilassa. Esimerkiksi: **`HGET hashKey field`**.
4.  Listan Toiminnot:
    -   Lisää elementti luetteloon: Käytä **`LPUSH`** tai **`RPUSH`** komentoa lisätäksesi elementin luettelon alkuun tai loppuun. Esimerkiksi: **`LPUSH listKey arvo`**.
    -   Nouda elementtejä luettelosta: Käytä **`LRANGE`** -komentoa noutaaksesi joukon elementtejä luettelosta. Esimerkiksi: **`LRANGE listKey start end`**.
5.  Aseta Toiminnot:
    -   Lisää elementtejä joukkoon: Käytä komentoa **`SADD`** lisätäksesi yhden tai useamman elementin joukkoon. Esimerkiksi: **`SADD setKey arvo1 arvo2`**.
    -   Nouda elementtejä sarjasta: Käytä komentoa **`SMEMBERS`** saadaksesi kaikki ryhmän jäsenet. Esimerkiksi: **`SMEMBERS setKey`**.
6.  Lajitellut Asetukset:
    -   Lisää elementtejä lajiteltuun ryhmään: Käytä **`ZADD`** -komentoa lisätäksesi elementtejä lajiteltuun joukkoon niiden pisteiden kanssa. Esimerkiksi: **`ZADD lajiteltuSetKey tulokset1 jäsen1 tulokset2 jäsen2`**.
    -   Nouda elementtejä lajitellusta sarjasta: Käytä **`ZRANGE`** -komentoa noutaaksesi joukon elementtejä lajitellusta sarjasta, joka perustuu niiden pisteisiin. Esimerkiksi: **`ZRANGE lajiteltuSetKey aloittaa loppu`**.

Huomaa: Voit viitata Redis-dokumentaatioon saadaksesi täydellisen listan komentoista ja niiden käytöstä.

## Käytä laatikkoa

Seuraavaksi osoitamme kuinka kartat tiedot `**Upstash**`sta `table`-komponenttiin.

#### Vaihe 1 Lisää toiminto

Muodostakaamme taulukon Upstashissa nimellä **`employee_profile`,** sisältäen 4 saraketta `name`, `level`, `age`, ja `naimisiin`.

Sitten voimme luoda uuden toiminnon Upstashille ILLA Cloudissa ja nimetä sen yläpuolelle2.

Jos haluat luetella kaikki tiedot employee_profile laita koodi snippet alla Redis kyselyn osiossa.

```sql
GET työntekijä_profiili
```

Koska tuotos tiedot on luettelo sanakirja pari, haluamme muuttaa sen taulukko. Kun muuntaja, ota se käyttöön ja kirjoita alla oleva koodi.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

![up_code](https://cdn.illacloud.com/official-website/img/docs/up_code.jpg)

Napsauta **`Tallenna`** ja **`Suorita`** aktivoidaksesi tämän toiminnon.

![up_data](https://cdn.illacloud.com/official-website/img/docs/up_data.jpg)
