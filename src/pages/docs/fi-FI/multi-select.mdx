## <Heading hidden>Monivalinta</Heading>

Monen valinnan komponentti voi olla hyödyllinen tilanteissa, joissa on valittava useita valintoja luettelosta. Tämä voi tehostaa tietojen syöttämistä ja suodatusta. Komponentti voidaan konfiguroida eri ominaisuuksia saavuttaa eri efektejä ja tukee myös menetelmiä ja tapahtumakäsittelijät enemmän räätälöintiä.

Voit lisätä vaihtoehtoja manuaalisesti tai irtotavarana tietolähteestäsi. Tässä artikkelissa esitellään yksityiskohtaisesti, miten käyttää usean valitse komponentti. 

### Ominaisuudet

#### Manuaaliset asetukset

Lisää uusia vaihtoehtoja tai poista asetukset manuaalisesti ja määritä asetukset yksitellen. Vaihtoehtojen ominaisuudet ovat seuraavat: 

| Ominaisuudet  | Kuvaus                                           |
| ------------- | ------------------------------------------------ |
| Tunniste      | The text displayed for the option                |
| Arvo          | Vaihtoehtoon liittyvä arvo                       |
| Pois Käytöstä | Onko valinta poistettu käytöstä valinnan varalta |

#### Kartoitetut asetukset

Voit myös lisätä vaihtoehtoja tietolähteestä liittämällä tunnisteen ja arvon ominaisuudet vastaaviin tietokenttiin. Voit saada vaihtoehtoja dynaamisesti tietokannasta tällä tavalla. Tämän artiklan lopussa osoitamme, miten sitä voidaan käyttää esimerkin avulla.

| Ominaisuudet  | Kuvaus                                                                     |
| ------------- | -------------------------------------------------------------------------- |
| Tietolähde    | Aseta valintojen tietolähde                                                |
| Tunniste      | Käytä {{item}} asettaaksesi valintojen otsikon                             |
| Arvo          | Käytä {{item}} valintojen arvon määrittämiseen                             |
| Pois Käytöstä | Käytä {{item}} määrittääksesi, onko valinta poistettu käytöstä valitaksesi |

### Menetelmä

Voit käyttää muita komponentteja komponentin ohjaamiseen. Kannatamme seuraavia kolmea menetelmää:

#### setValue

Valitun vaihtoehdon asettamiseksi esimerkiksi {{[”arvo1”,”arvo3”…]}}

| Ominaisuudet | Kuvaus                                     |
| ------------ | ------------------------------------------ |
| Arvo         | Valikoitujen vaihtoehtojen arvojen sarja.  |

#### clearValue

Poista valitut vaihtoehdot

#### validate

Varmistetaan, että syöttötiedot ovat laillisia

#### kirjoita

Vahvistusviestin tyhjentämiseksi

### Tapahtuman käsittelijä

Tuemme kuuntelemista onChange tapahtumasta, joka on monivalinta komponentti.

### Tiedot

| Ominaisuuden nimi | Kuvaus                            |
| ----------------- | --------------------------------- |
| arvo              | joukko valittuja optioiden arvoja |

### Käytä laatikkoa

Seuraavaksi osoitamme, miten kartoitettua tilaa voidaan käyttää saadaksesi dynaamisesti vaihtoehtoja. 

#### Vaihe 1 Lisää toiminto

Olemme luoneet taulun nimeltä `selection`, mukaan lukien 5 saraketta: `options_en`, `options_jp`, `options_zh`, `options_kr`, `value`. Tallensimme valitsimen arvot `value`-muotoon ja tallennamme asetusmerkin eri kielillä muihin sarakkeisiin. 

Luo toiminnot luetteloidaksesi kaikki tiedot `selection`-nimestä `postgresql1`

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/multiselect1.png' alt="" />

```jsx
valitse * valinnasta
```

#### Vaihe 2 Määritä monivalinta komponentti

1.  Set the Data sources to `{{postgresql1.data}}` 
2.  Määritä otsikko muuttaaksesi otsikkoa kielen perusteella. 

    1.  Aseta sarake käyttämällä {{ item.columnName }}. 
    2.  Käytä {{ currentUserInfo.language }} asettaaksesi kielen käyttäjät käytettäväksi ILLA:ssa.

    ```jsx
    {{currentUserInfo.language=='ja-JP' ? item.options_jp 
    : currentUserInfo.language == 'ko-KR' ? item.options_kr 
    : currentUserInfo.language == 'zh-CN' ? item.options_zh 
    : item.options_en }}
    ```
