## <Heading hidden>Mukautettu API todennus</Heading>

ILLA tukee monia autentikointistandardeja oletuksena (katso tästä lisätietoja). 
Lisäksi, ILLA on Custom Auth autentikointitila, jonka avulla voit yhdistää useita vaiheita tukee lähes mitä tahansa API todentaminen. 
Näitä todennusmenetelmiä voidaan käyttää sekä REST että GraphQL resursseja.
