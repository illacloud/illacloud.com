## <h2 hidden>Wat is een datumbereik?</h2>

Datumbereik component is een UI component waarmee gebruikers een reeks data kunnen selecteren uit een kalender interface, Doorgaans gebruikt in formulieren of gegevensinvoerinterfaces waar de gebruiker een start- en einddatumbereik moet opgeven.

Wanneer een datumbereik component aan een canvas wordt toegevoegd, toont het twee agenda-interfaces naast elkaar. EÃ©n kalender interface wordt gebruikt om de startdatum te selecteren, de andere wordt gebruikt om de einddatum te selecteren. De gebruiker kan door de kalenders navigeren door op de vorige of volgende pijlen te klikken om naar een andere maand te gaan of door te klikken op een specifieke datum om deze te selecteren. De geselecteerde datums worden weergegeven in de invoervelden.

Om de begindatumwaarde in te stellen, bijvoorbeeld {{"value1"}}

### Gebruik geval

Vervolgens zullen we laten zien hoe we de data van de gegevensbron kunnen koppelen aan het datumbereik en de eindwaarde kunnen instellen op de datum van vandaag met een knop. 

#### Stap 1 Voeg een actie toe

Laat ons een tabel aanmaken in REST API genaamd **`promoteCodes`,** inclusief 6 kolommen: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. We zullen het startedAt en verlopen voor voorbeeldgebruik. 

Dan kunnen we een nieuwe actie voor REST API maken uit de actielijst, waarin alle gegevens in promoteCodes en restapi1 worden weergegeven. 

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

#### Stap 2 Componenten toevoegen

Vervolgens kunnen we een **`date range`** component en een **`button`** component toevoegen aan het canvas. Voor de standaard start datum waarde van het datumbereik component, gebruiken we de startOp tijd van de eerste promotiecode van de rest van de api gegevens.

```sql
{{restapi1.data.promoteCodes[0].startedAt}}
```

Evenzo gebruiken we voor standaard einddatum waarde VervaldatumOp het moment van de eerste promotiecode van de rest van de api data.

```sql
{{restapi1.data.promoteCodes[0].expiredAt}}
```

We noemden ook de knop als 'Update' zoals hieronder weergegeven.

![date_range_0](https://cdn.illacloud.com/official-website/img/docs/date_range_0.png)

#### Stap 3 Configureer het component

Voor het **`button`** component kunnen we het configureren om de einddatum waarde in **`date bereige`** op de datum van vandaag in te stellen.

1.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, selecteer de **`date range`** component die u wilt bijwerken als doel van het event.
2.  Kies de **`setEndValue`** actie en zet de **`End date`** als `{{currentUserInfo.updatedAt}}` 

![date_range_config](https://cdn.illacloud.com/official-website/img/docs/date_range_config.png)

#### Stap 4 test

Nu wanneer u op de knop 'Bijwerken' klikt, moet de einddatum worden gewijzigd naar de datum van vandaag. We hebben het hier over 2023-5-6, maar het kan anders zijn als je het testen.
