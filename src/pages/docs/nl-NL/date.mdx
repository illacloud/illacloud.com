## <h2 hidden>Wat is datum?</h2>

Het datumcomponent is een component waarmee gebruikers een datum uit een kalenderinterface kunnen selecteren. Het datumcomponent wordt meestal gebruikt in formulieren of interfaces voor gegevensinvoer waarbij de gebruiker een specifieke datum moet selecteren.

Wanneer een datumcomponent aan het canvas wordt toegevoegd, toont het standaard een kalenderinterface. De gebruiker kan door de kalender navigeren door op de vorige of volgende pijlen te klikken om naar een andere maand te gaan of door te klikken op een specifieke datum om deze te selecteren. De geselecteerde datum zal worden weergegeven in het invoerveld.

### Methods

U kunt andere componenten gebruiken om het onderdeel te beheren. Wij steunen de volgende vier methoden:

-   **setvalue**

Om de invoerdatum in te stellen, bijvoorbeeld {{'value1'}}

| Eigenschappen | Beschrijving |
| ------------- | ------------ |
| Waarde        | Input value  |

-   **clearvalue**

Om de waarde van het geselecteerde component te wissen 

-   **validate**

Om te controleren of de invoerinformatie legaal is

-   **clearvalidatie**

Om het validatiebericht te wissen

### Gebruik geval

Vervolgens zullen we laten zien hoe we de gegevens van de databron tot nu toe kunnen koppelen en instellen op huidige datum met een knop. 

#### Stap 1 Voeg een actie toe

Laat ons een tabel aanmaken in REST API genaamd **`promoteCodes`,** inclusief 6 kolommen: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. We zullen de bijgewerkte versie op tijd gebruiken voor voorbeeldgebruik. 

Dan kunnen we een nieuwe actie voor REST API maken uit de actielijst, waarin alle gegevens in promoteCodes en restapi1 worden weergegeven. 

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

#### Stap 2 Componenten toevoegen

Vervolgens kunnen we een **`date`** component en een **`button`** component toevoegen aan het canvas. Voor de standaardwaarde van het datumcomponent, hebben we het ingesteld als de laatste updatetijd van de eerste promotiecode van de rest van de api gegevens.

```sql
{{restapi1.data.promoteCodes[0].updatedAt}}
```

We hebben de knop ook gelabeld als "Vandaag" zoals hieronder weergegeven

![date_time0](https://cdn.illacloud.com/official-website/img/docs/date_time0.png)

#### Stap 3 Configureer het component

Voor het **`button`** component kunnen we het configureren om de datum in het **`date`** onderdeel op de datum van vandaag in te stellen.

1.  In de Event Handler bewerken, selecteer **`Control component`** in actie, selecteer het **`date`** onderdeel dat u wilt bijwerken als het doel van het event.
2.  Kies de **`setValue`** actie en stel de waarde in als `{{currentUserInfo.updatedAt}}` 

![date_confg](https://cdn.illacloud.com/official-website/img/docs/date_confg.png)

#### Stap 4 test

Nu wanneer je op de knop "vandaag" klikt, moet de tijd veranderen naar de datum van vandaag. We hebben het hier over 2023-5-6, maar het kan anders zijn als je het testen.
