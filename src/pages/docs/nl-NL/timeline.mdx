## <h2 hidden>Wat is tijdlijn?</h2>

Een tijdlijncomponent is een grafische weergave van gebeurtenissen of gegevenspunten gerangschikt chronologisch langs een horizontale of verticale as. Het stelt gebruikers in staat om te visualiseren en te navigeren door een reeks evenementen of mijlpalen op een lineaire manier.

## Gebruik geval

Vervolgens zullen we laten zien hoe we de data van de data bron kunnen toewijzen aan de tijdlijn component.

#### Stap 1 Voeg een actie toe

Laat ons een tabel maken in Supabase genaamd **`timeline`,** inclusief slechts 1 kolommen: mijlpalen. Er zijn 4 mijlpalen die we in kaart willen brengen naar de tijdlijn component.

Dan kunnen we een nieuwe actie voor Supabase van de actielijst genaamd supabasedb1 maken.

Om alle gegevens in mijlpaal weer te geven, zet u de code snippet hieronder in het SQL query gedeelte.

```sql
selecteer * uit tijdlijn;
```

Aangezien de uitvoergegevens een woordenboek zijn, willen we het veranderen in een array. Activeer het in Transformatie en zet de code hieronder in.

```jsx
const getransformedData = data.map(obj) => Object.values(obj)[0]);
retourneer transformedData;
```

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### Stap 2 Componenten toevoegen

Vervolgens kunnen we een **`timeline`** component toevoegen aan de canvas.

In het `**Items**` veld op de Inspect pagina nadat je op het onderdeel hebt geklikt, zet{{supabasedb1.data}}\` om de gegevens op te halen uit supabase.
