## <Heading hidden>Autenticazione API personalizzata</Heading>

ILLA supporta molti standard di autenticazione per impostazione predefinita (vedi qui per maggiori informazioni). 
Inoltre, ILLA ha una modalità di autenticazione Auth Personalizzata che consente di combinare più passaggi che supportano praticamente qualsiasi autenticazione API. 
Questi metodi di autenticazione possono essere utilizzati sia per le risorse REST e GraphQL.
