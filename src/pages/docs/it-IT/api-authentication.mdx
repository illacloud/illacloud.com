## <Heading hidden>Autenticazione API</Heading>

ILLA può supportare quasi tutti i tipi di autenticazione. Abbiamo collaborato con varie aziende per supportare tutti i tipi standard di autenticazione e abbiamo anche sviluppato metodi per supportare qualsiasi metodo di autenticazione personalizzato che non supportiamo nativamente.

È possibile utilizzare i token di autenticazione statica o creare un sistema di autenticazione che richiede all'utente finale per le proprie credenziali ogni volta che accede all'applicazione.

Questa pagina ti fornirà le istruzioni su come autenticare la tua API. Se hai domande o hai bisogno di aiuto, è possibile contattare il nostro team di supporto utilizzando la bolla di interfono su questa pagina o l'icona di aiuto all'interno dell'editor.
