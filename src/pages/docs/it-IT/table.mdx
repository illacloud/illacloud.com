#### Passo 1 Aggiungere un'azione

Creiamo una tabella in Upstash chiamata **`employee_profile`,** che include 4 colonne `name`, `level`, `age`, e `sposato`.

Quindi possiamo creare una nuova azione per Upstash dall'elenco di azioni in ILLA Cloud e chiamato upstash2.

Per elencare tutti i dati nel profilo dipendente_inserire la snippet di codice qui sotto nella sezione Ricerca Redis.

```sql

```

PoichÃ© i dati di output sono un elenco di coppie di dizionari, vogliamo trasformarli in un array. In Transformer, abilitarlo e mettere giÃ¹ il codice qui sotto.

```jsx

// Parse the JSON string into a JavaScript object

// Access the "employees" property and map it to a new array

  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```
