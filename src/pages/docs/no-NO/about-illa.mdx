## <Heading hidden>om ILLA</Heading>

ILLA lar deg bygge og distribuere interne apper raskt. Koble til databaser og APIer, og koble til brukergrensesnitt med dra-og-slipp blokker som tabeller og skjemaer, og skriv spørringer for å samhandle med data ved hjelp av SQL og JavaScript.

### Hva er ILLA?

ILLA er en plattform med åpen kildekode som kan bygges opp og bruke interne verktøy utrolig fort.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png'
  alt=""
/>

Med ILLA kan du enkelt koble til ulike datakilder som databaser, API/GraphQL endepunkter, SaaS-verktøy og tjenester for nettlagring og nettjenester. Dette inkluderer populære alternativer som PostgreSQL, MongoDB, MySQL, Stripe, Google Sheets og AWS S3. Når du er koblet til kan ILLA kjøre spørringer for å hente og oppdatere data fra disse kildene. Brukergrensesnittet (UI) gir praktiske komponenter for visualisering og modifisering av data, som tabeller, kart og former.

Med ILLA, kan du raskt og enkelt bygge et brukergrensesnitt (UI) ved hjelp av forhåndbygde widgets som du kan dra og slippe til et rutenett stil lerretet. Dette gjør det enklere å integrere programmet front-end og back-end og optimalisere byggeprosessen. I tillegg støtter ILLA JavaScript i widgets, spørringer og andre komponenter, slik at du kan legge til logiske, transformeringsdata og definere komplekse arbeidsprosesser. Dette gjør det enkelt å tilpasse søknaden din etter dine spesifikke behov.
