## <Heading hidden>Gráfico</Heading>

### Mostrar datos en un gráfico

Puede utilizar acciones y transformadores para construir gráficos. Puedes hacer referencia a un array u objeto en el campo **Fuente de datos** para un gráfico. 

Después de seleccionar una fuente de datos, ILLA analiza los datos y rellena el gráfico. Por defecto, la primera columna no numérica rellena el **eje X** pero puedes cambiar esta y otras propiedades como **tipo de cartas**, **Grupo por**, etc.
