## <Heading hidden>Lista de integración</Heading>

ILLA ofrece una gama de integraciones preconstruidas que le permiten conectarse fácilmente a sus fuentes de datos con una configuración mínima. Simplemente seleccione la integración apropiada de la lista de opciones disponibles, introduzca las credenciales necesarias, y puede comenzar a utilizar su fuente de datos dentro de ILA. Esto elimina la necesidad de configurar configuraciones complejas y le permite comenzar a trabajar rápidamente con sus datos.

### Integraciones de base de datos

| Nombre            | Descripción                                                                                                                                                                                                                                                    |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql             | MySQL es un sistema de gestión de bases de datos relacionales (RDBMS) de código abierto basado en el lenguaje de consulta estructurada (SQL).                                                                                                                  |
| PostgreSQL        | PostgreSQL es un sistema de gestión de bases de datos relacional con objetos de código abierto (ORDBMS) conocido por su fiabilidad y riqueza de características.                                                                                               |
| MariaDB           | MariaDB es un sistema de gestión de bases de datos relacionales de código abierto que es un fork de la base de datos MySQL.                                                                                                                                    |
| TiDB              | TiDB es una base de datos NewSQL de código abierto que soporta cargas de trabajo de procesamiento analítico y transaccional de Hybrid.                                                                                                                         |
| Redis             | Redis es un almacén de estructura de datos de código abierto en memoria que a menudo se utiliza como una base de datos, caché y broker de mensajes.                                                                                                            |
| MongoDB           | MongoDB es un programa de bases de datos multiplataforma gratuito y de código abierto orientado a documentos.                                                                                                                                                  |
| Búsqueda elástica | Elasticsearch es un motor de búsqueda y análisis distribuido de código abierto construido sobre Apache Lucene.                                                                                                                                                 |
| Firebase          | Firebase es una plataforma de desarrollo de aplicaciones móviles y web basada en la nube desarrollada por Google.                                                                                                                                              |
| Supabase DB       | Supabase es una plataforma de código abierto para construir aplicaciones sin servidor en tiempo real. Está construido sobre PostgreSQL y proporciona una serie de herramientas y servicios para construir y desplegar rápidamente aplicaciones sin servidores. |
| Clickhouse        | ClickHouse es un sistema de gestión de bases de datos orientado a columnas de código abierto que está diseñado para procesamiento analítico en línea (OLAP) y análisis de datos en tiempo real.                                                                |

### Integraciones de API

| Nombre                | Descripción                                                                                                                                                      |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| API REST              | REST es un estilo arquitectónico para la construcción de servicios web. Las APIs RESTful son APIs que siguen el estilo arquitectónico REST.                      |
| Amazon S3(General S3) | Amazon S3 es un servicio de almacenamiento de objetos ofrecido por AWS. Proporciona almacenamiento seguro, escalable y de bajo costo para los datos en Internet. |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) es un protocolo para enviar y recibir mensajes de correo a través de Internet.                                              |
| Gráfica               | GraphQL es un lenguaje de consulta para APIs y un tiempo de ejecución para cumplir esas consultas con los datos existentes.                                      |
