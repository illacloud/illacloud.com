Com nossa extensa biblioteca de componentes de UI, você pode rapidamente e facilmente criar interfaces altamente polidas. Você pode escrever consultas para interagir com os dados e usar JavaScript em muitas partes diferentes do seu aplicativo.

## Montar componentes

Use a biblioteca de componentes IU da ILLA, para criar a interface de usuário do seu aplicativo.

Componentes são uma parte essencial do ILLA, e eles funcionam da mesma forma que componentes React que podem aceitar propriedades e ter estado interno. Ao combinar a biblioteca de componentes com seus dados, você poderá criar aplicativos interativos.

Alguns componentes comuns incluem:

-   O componente Table é usado para exibir dados em um formato tabular e oferece funcionalidades como ordenação, filtragem, paginação e exportações CSV.
-   O componente de Formulário é usado para agrupar campos de entrada e submetê-los como uma única unidade.
-   O componente de seleção é uma combinação de um menu suspenso e um campo de entrada que permite aos usuários selecionar um valor a partir de uma lista ou digitar um valor diretamente como texto.
-   O componente Gráfico é usado para criar visualizações de dados usando gráficos e gráficos.
