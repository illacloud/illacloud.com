## <Heading hidden>Sobre a ILLA</Heading>

ILLA permite que você crie e faça deploy de aplicativos internos rapidamente. Conecte-se aos seus bancos de dados e APIs, UIs de montagem com blocos de construção de arrastar e soltar, como tabelas e formulários, e escrever consultas para interagir com dados usando SQL e JavaScript.

### O que é ILLA?

ILLA é uma plataforma de código baixo para desenvolvedores construírem e implementarem ferramentas internas com notável rapidez.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png'
  alt=""
/>

Com a ILLA, você pode facilmente conectar várias fontes de dados, tais como bancos de dados, endpoints API/GraphQL, ferramentas SaaS e serviços de armazenamento em nuvem. Isto inclui opções populares como PostgreSQL, MongoDB, MySQL, Stripe, Google Sheets, AWS S3. Uma vez conectado, ILLA pode executar consultas para buscar e atualizar dados destas fontes. A interface do usuário (UI) fornece componentes convenientes para visualizar e modificar os dados, como tabelas, gráficos e formulários.

Com a ILLA, você pode rapidamente e facilmente construir uma interface de usuário (UI) usando widgets pré-construídos que pode arrastar e soltar para uma tela ao estilo grade. Isso simplifica a integração do front-end e back-end da sua aplicação e otimiza o processo de construção. Além disso, ILLA suporta JavaScript dentro de widgets, consultas e outros componentes, permitindo que você adicione lógica, transforme dados e defina fluxos de trabalho complexos. Isso facilita a customização do seu aplicativo para suas necessidades específicas.
