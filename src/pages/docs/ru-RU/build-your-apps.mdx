## <Heading hidden>Создайте ваши приложения</Heading>

Благодаря нашей обширной библиотеке компонентов пользовательского интерфейса вы можете быстро и легко создавать высокополированные интерфейсы. Вы можете написать запросы, чтобы взаимодействовать с данными и использовать JavaScript во многих различных частях вашего приложения.

### Редактор

Создание, развертывание и запуск приложений с помощью веб-среды разработки ILA

### Дизайн и удобство использования (страницы, преобразователи)

Персонализируйте приложения с помощью пользовательских стилей и форматирования

### Обработчики событий

Обработчики событий позволяют вам запускать запросы, контролировать компоненты и выполнять другие действия в ответ на взаимодействие пользователя

## Компоненты сборки

Используйте библиотеку пользовательского интерфейса ILLA, чтобы создать пользовательский интерфейс вашего приложения.

Компоненты являются неотъемлемой частью ILLA, и они функционируют аналогично React компонентов, в которых они могут принимать свойства и иметь внутреннее состояние. Объединяя библиотеку компонентов ILA с вашими данными, вы можете создавать интерактивные приложения.

К некоторым общим компонентам относятся:

-   Компонент таблицы используется для отображения данных в виде таблицы, и предлагает такие функции, как сортировка, фильтрация, пагинация и экспорт CSV.
-   Компонент формы используется для группировки полей ввода и отправки их в одну единицу.
-   Входной компонент Select - это комбинация выпадающего списка и поля ввода, которые позволяют пользователям либо выбрать значение из списка, либо ввести значение непосредственно в виде текста.
-   Компонент диаграммы используется для визуализации данных с помощью графиков.
