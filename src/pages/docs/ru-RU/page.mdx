## <Heading hidden>Страница</Heading>

Многостраничная демо-версия, построенная на ILLA: 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page1.gif' alt="" />

Одностраничное приложение - это приложение, которое может изменять страницы без перезагрузки всех компонентов на странице. Большинство ресурсов (включая HTML, CSS и Script) загружаются только один раз в течение всего жизненного цикла приложения, что делает быстрое одностраничное приложение.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page2.png' alt="" />

В многостраничном приложении, каждое изменение такие как отображение данных или отправка данных обратно на сервер запросов рендеринга новой страницы с сервера. Многостраничное приложение имеет несколько уровней, что делает структуру приложения простой и простой для пользователей.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page3.png' alt="" />

В ILLA Builder одностраничное приложение может быть построено путем установки рамок страницы и добавления вида в разделы, и многостраничное приложение может быть построено путем добавления нескольких страниц, а разработчики могут использовать оба этих шаблона дизайна в одном приложении.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page4.png' alt="" />

### Установить макет страниц

Перейдите к настройке страницы в следующих шагах：

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page5.png' alt="" />

Вы можете использовать предустановленные макеты или вручную добавлять и удалять разделы, чтобы задать макет. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page6.gif' alt="" />

### Добавить вид на страницу

`Просмотр пути доступа`: строка, созданная после URL страницы для доступа к указанному представлению. После развертывания приложения полный адрес доступа выглядит следующим образом: https://builder.illacloud.com/{{team_identifier}}/deploy/app/{{app_id}}/{{page_name}}/{{view_path}}

Вы также можете ввести домашнюю страницу и вид по умолчанию по следующему пути:

https://builder.illacloud.com/{{team_identifier}}/deploy/app/{{app_id}}. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page7.png' alt="" />

### Подключить меню и страницы

Добавить обработчики событий в пункты меню → установить `Действие` в `Установить маршрутизатор` → выбрать страницу и просмотреть
