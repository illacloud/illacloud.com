## <Heading hidden>О программе</Heading>

ILLA позволяет быстро создавать и развертывать внутренние приложения. Подключайтесь к вашим базам данных и API, собирайте пользовательские интерфейсы с помощью перетаскиваемых блоков, таких как таблицы и формы, и напишите запросы для взаимодействия с данными с помощью SQL и JavaScript.

### Что такое ILLA?

ILLA является открытым исходным кодом низкокодовой платформой для разработчиков для создания и развертывания внутренних инструментов удивительно быстро.

С помощью ILLA вы легко можете подключаться к различным источникам данных, таким как базы данных, конечные точки API/GraphQL, SaaS инструменты и облачные сервисы. К ним относятся такие популярные опции, как PostgreSQL, MongoDB, MySQL, Stripe, Google Sheets, и AWS S3. ILLA после подключения может запускать запросы для получения и обновления данных из этих источников. Интерфейс пользователя (UI) предоставляет удобные компоненты для визуализации и изменения данных, таких как таблицы, диаграммы и формы.

С помощью ILLA вы можете быстро и легко создать пользовательский интерфейс (UI) с помощью встроенных виджетов, которые можно перетащить на холст в табличном стиле. Это упрощает интеграцию внешнего и внутреннего приложения и оптимизирует процесс строительства. Кроме того, ILLA поддерживает JavaScript внутри виджетов, запросов и других компонентов, позволяя добавлять логику, преобразовывать данные и определять сложные рабочие процессы. Это упрощает настройку вашего приложения в соответствии с вашими потребностями.
