## <Heading hidden>JavaScript гидер</Heading>

JavaScript является самым важным компонентом ILLA Builder, , который вы используете для создания доступа к данным и создания логики вашего приложения. Освоение вызова JavaScript, вы освоили использование ILLA Builder.

При создании вашего приложения в ILLA Builder, вы можете использовать `{{template syntax}}` для написания пользовательских операторов JavaScript для вызова и управления данными, или напишите код JS прямо в преобразователе.

### Данные операции

#### Доступ к данным по объектам

В ILLA Builder вы можете вводить выражения JavaScript в любое текстовое поле с помощью синтаксиса шаблона. Объекты данных ссылаются, и вы можете манипулировать данными всех манипуляций, действий и трансформаций с помощью JavaScript. Например, вы можете использовать `{{textInput.data}}`, чтобы получить доступ к содержимому вставки текста, `{{action.data}}`, чтобы получить доступ к данным из базы данных действий и `{{transformer.value}}`, чтобы получить доступ к данным из трансформатора.

Мы добавили и изменили зеркало кода вставить в продукт, адаптировала его к среде программирования ILLA Builder, поддерживала подсветку синтаксиса и автоматическое отступление, добавил подробное описание переменных и функций, которые возникают во время программирования, чтобы вы могли научиться использовать ILLA Builder при низкой стоимости. После того, как вы ввели имя объекта, введите точку. Заявка будет содержать все свойства объекта, тип данных и частичное текстовое описание, , чтобы идентифицировать объект, к которому можно получить доступ в кратчайшие сроки.

#### Доступ к номерам в массивах

Когда вам нужно получить доступ к значению в определенном массиве, вы можете получить значение по местоположению или подписке данных, к которым вам необходим доступ.

```javascript
const list = ['a', 'b', 'c']
список возвратов[0] // Возврат первого значения в списке 'a'
```

### Просмотр данных

Для просмотра данных в Builder ILLA существует четыре раздела: рабочая область данных в левой панели, запрос данных ниже текстового поля после ввода выражения JavaScript, и отображение данных через вставку данных для отображения данных.

#### Представление рабочей области данных

Вы можете найти рабочую область данных, открыв левую панель в редакторе приложений, так что данные представляются как структурированные JSON, в которых каждый объект и список могут быть расширены, а все данные визуально отображаются в рабочей области.

[//]: # "[图片]"

#### Предпросмотр ввода данных в реальном времени

Когда вы вызываете действие или трансформатор в поле ввода, данные для этой операции автоматически всплывают под поле ввода, и вы можете подтвердить, что вызванные данные верны на основе информации здесь.

[//]: # "[图片]"

#### Представление данных

Вставка данных дает вам наиболее интуитивно понятное представление данных. Просто перетащите компонент в рамку, введите имя соответствующего источника данных в поле данных и упорядочить, и у вас будет доступ к информации о отображении данных.

[//]: # "[图片]"

### Трансформатор

Вы можете писать несколько строк JS кода в преобразователе, и мы рекомендуем вам писать в трансформаторе, а не в двойных фигурных скобках. Вы можете запрограммировать JavaScript непосредственно в трансформаторе и вызвать данные в операции `{{}}`.
