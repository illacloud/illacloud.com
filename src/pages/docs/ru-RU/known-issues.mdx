### Проблема типа монтирования

Запустите `illa deploy --self -port=1000 -s={your-ip-address}` или `run-official-image.sh`, возвращая следующую ошибку:

`Docker ответил статус-кодом 400: недопустимый mount config для типа "bind": bind source path не существует: /tmp/illa-data`

На данный момент тип монтирования поддерживает только `mount`, пожалуйста настройте `docker -v` в режиме `mount`. Или переустановите docker следующими скриптами.

[reinstall-docker-with-ubuntu.sh](https://github.com/illacloud/deploy-illa-manually/blob/main/utils/reinstall-docker-with-ubuntu.sh)

И запустите команду развертывания еще раз.

Для других проблем: https://github.com/illacloud/illa-builder/issues
