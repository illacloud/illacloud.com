## <Heading hidden>Установка введение</Heading>

### Облако

Попробуйте установить систему в облаке ILLA одним кликом.

### Само-хост

Вы можете самостоятельно владеть локально с ILLA CLI, Docker композиция, k8 шлем и т.д. Мы многократно оптимизировали процесс самостоятельного хостинга, чтобы позволить вам установить самохост и программное обеспечение всего за несколько минут. По сравнению с облаком, вы можете безопаснее и надежнее использовать продукты ILLA, и вы можете использовать ILLA Builder, просто посещая сайт установки после хостинга и развертывания строителя локально.

Дополнительную информацию о самохостинге см. в ILA CLI, Docker compose, k8s загрузочных страницах шлема.
Если у вас возникли трудности с самостоятельным развертыванием, пожалуйста, свяжитесь с нами по адресу GitHub или business@illasoft.com.
