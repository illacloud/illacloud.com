## <Heading hidden>API автентифікація</Heading>

ILLA може підтримувати майже будь-який вид аутентифікації. Ми співпрацюємо з різними компаніями для підтримки всіх стандартних типів аутентифікації і також розробляли методи для підтримки будь-яких методів користувацької аутентифікації, які ми не підтримуємо належним чином.

Ви можете використовувати статичні токени автентифікації або створити систему аутентифікації, яка пропонує кінцевого користувача для своїх облікових даних кожного разу, коли вони отримують доступ до додатку.

Ця сторінка надасть вам інструкції для аутентифікації вашого API. Якщо у вас є питання або потрібна допомога, ви можете зв'язатися з нашою командою підтримки за допомогою міхура Intercom на цій сторінці або піктограму довідки всередині редактора.
