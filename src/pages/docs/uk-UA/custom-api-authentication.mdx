## <Heading hidden>Автентифікація</Heading>

ILLA підтримує багато стандартів автентифікації за замовчуванням (див. тут для отримання додаткової інформації). 
Додатково, ILLA має режим автентифікації автентифікації користувача що дозволяє вам комбінувати кілька кроків, що підтримують будь-яку API автентифікацію. 
Ці методи автентифікації можуть бути використані для REST і GraphQL ресурсів.
