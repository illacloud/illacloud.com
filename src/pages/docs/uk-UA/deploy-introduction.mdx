## <Heading hidden>Розгортання</Heading>

### Хмара

Спробуйте розгорнути в Cloud одним кліком для найпростішого враження.

### Самоприймач

Ви можете самостійно хостинг локально з ILLA CLI, композом Docker, шолом і т.д. Ми неодноразово оптимізували процес самогосподарства, щоб дозволити вам робити самохост і програмне забезпечення для впровадження всього за кілька хвилин. Порівняно з хмарою, ви можете використовувати ILLA продукти більш надійно та надійно, ви можете використовувати конструктору ILLA Builder, просто відвідавши сайт розгортання після проведення хостингу та розгорнувши конструктор на локальному рівні.

Для отримання додаткової інформації про себе, будь ласка, дивіться ILA CLI, Docker compose, сторінки бук-шолому.
Якщо у вас виникли труднощі з самостійним розгортанням, зв'яжіться з нами на GitHub або business@illasoft.com.

#### Розгорнути ILLA CLI

Будь ласка, перевірте: [ILLA CLI](https://www.illacloud.com/docs/illa-cli)
