## <Heading hidden>Список інтеграцій</Heading>

ILLA пропонує широкий спектр попередньо побудованих інтеграцій, що дозволяє легко з'єднатися з джерелами даних з мінімальною конфігурацією. Просто виберіть відповідну інтеграцію зі списку доступних варіантів, введіть необхідні облікові дані, і ви можете почати використовувати ваші дані в межах ILLA. Це усуває потребу в налаштуванні складних конфігурацій і дозволяє швидко почати роботу з вашими даними.

### Інтеграція бази даних

| Ім'я             | Опис                                                                                                                                                                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql            | MySQL - це система керування базами даних з відкритим вихідним кодом (RDBMS) заснована на мові структурованого запиту (SQL).                                                                                                          |
| PostgreSQL       | PostgreSQL - це система управління базами даних з відкритим вихідним кодом (ORDBMS) яка відома своєю надійністю та багатістю функцій.                                                                                                 |
| MariaDB          | MariaDB - це реляційна система управління базами даних з відкритим вихідним кодом, яка є форком бази даних MySQL.                                                                                                                     |
| ТіБД             | TiDB - це база даних з відкритим вихідним кодом NewSQL, яка підтримує Hybrid Transactional і аналітичне завантаження процесів.                                                                                                        |
| Редіс            | Крамниця даних з відкритим вихідним кодом яка часто використовується як база даних, кеш та брокер повідомлень.                                                                                                                        |
| MongoDB          | MongoDB є безкоштовною і відкритим вихідною програмою з відкритим вихідним кодом для створення баз даних.                                                                                                                             |
| Еластичний пошук | Elasticsearch це відкритий вихідний код, розподілений пошуковий та аналітичний двигун, побудований на вершині Apache Lucene.                                                                                                          |
| Firebase         | Firebase - це хмарна мобільна платформа та веб-застосунків, розроблена компанією Google.                                                                                                                                              |
| Supabase DB      | Supabase — це платформа з відкритим вихідним кодом для збирання безсерверних програм у реальному часі. Він побудований поверх PostgreSQL і надає ряд інструментів і сервісів для швидкого створення та розгортання серверних програм. |
| Clickhouse       | Кліклюз - це система управління базами даних з відкритим вихідним кодом, розроблена для онлайн-аналітичної обробки (OLAP) і аналізу даних у реальному часі.                                                                           |

### API інтеграції

| Ім'я                   | Опис                                                                                                                            |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| REST API               | REST - це архітектурний стиль для будівельних веб-сервісів. RESTful API-це те, що слідують за РЕШТ-архітектурним стилем.        |
| Amazon S3(General S3)  | Amazon S3 - це сховище об'єктів, яке пропонує AWS. Він забезпечує безпечний, масштабований та дешевий носій для даних в мережі. |
| Сервер вихідного листа | SMTP (простий протокол передавання пошти) - це протокол для відправки та отримання листів через Інтернет.                       |
| Графік QL              | GraphQL - це мова запиту для API і робочий час для виконання цих запитів з вашими існуючими даними.                             |
