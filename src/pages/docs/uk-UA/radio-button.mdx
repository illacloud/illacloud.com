## <Heading hidden>Радіо кнопка</Heading>

Радіокнопка - вибір компоненту введення, який дозволяє користувачам вибирати один варіант групи попередньо визначених варіантів. Він виглядає як закруглена квадратна кнопка, яка може бути обрана користувачем або знебір. Коли вибрана кнопка радіо, всі інші радіо-кнопки в одній групі будуть автоматично зняті, забезпечуючи, що одночасно можна вибрати тільки один варіант. Радіокнопки зазвичай використовуються в формах, анкетах, а також опитуваннях, що допомагають користувачам робити окремий вибір із списку варіантів.

### Властивості

#### Ручні налаштування

Додайте нові опції або видаліть параметри вручну і налаштуйте параметри один за одним. Властивості варіантів наступним чином: 

| Властивості | Опис                           |
| ----------- | ------------------------------ |
| Мітка       | Текст, відображений для опції  |
| Цінність    | Значення, пов'язане з опцією   |
| Вимкнено    | Чи відключена опція для вибору |

#### Налаштування зіставлених

Ви також можете додати опції з джерела даних, натиснувши на мітки та значення у відповідних полях даних. Ви можете отримати параметри динамічно з бази даних. В кінці цієї статті ми продемонструємо, як використовувати її в прикладі.

| Властивості   | Опис                                                                   |
| ------------- | ---------------------------------------------------------------------- |
| Джерело даних | Встановити джерело даних опцій                                         |
| Мітка         | Використовуйте {{item}} , щоб встановити мітку опцій                   |
| Цінність      | Використовуйте {{item}} , щоб встановити значення опцій                |
| Вимкнено      | Використовуйте {{item}} , щоб встановити, чи вимкнена опція для вибору |

### Метод

Ви можете використовувати інші компоненти для керування компонентами. Ми підтримуємо наступні три методи:

#### значення набору

Наприклад, щоб встановити вибраний параметр, наприклад, {{”value1”}}

| Властивості | Опис                             |
| ----------- | -------------------------------- |
| Цінність    | Рядок вибраного значення опції.  |

#### чисте значення

Очистити вибрані параметри

#### validate

Щоб переконатися, що вхідна інформація є правовою

#### очистити усі

Щоб очистити повідомлення підтвердження

### Обробник подій

Ми підтримуємо слухаємо подію `onChange` компонента радіокнопки. 

Якщо вибраний параметр зміниться, він може викликати певну дію. Це означає, що коли користувач вибирає інший варіант групи радіоклавіш, можна вжити конкретної дії на основі вибраної опції. Наприклад, у веб-формі, обирається інша опція, може викликати відображення різних полів форми або подання форми до іншого призначення. Таким чином, вибір користувача може впливати на поведінку програмного забезпечення чи веб-додатку, яку він використовує. Можливість викликати вказані дії на основі вводу користувача є важливим аспектом інтерактивної розробки програмного забезпечення і може покращити зручність і функціональність програмного забезпечення.

### Використати регістр

Далі, ми продемонструємо, як використовувати відображений режим для динамічного отримання варіантів.

Створіть дії для списку всіх даних в `selection` з ім'ям `postgresql1`


#### Крок 2 налаштування компонента

1.  Встановити джерела даних на `{{postgresql1.data}}` 
2.  Налаштуйте етикетку, щоб змінити мітку на основі мови. 

    1.  Використовувати {{ item.columnName }} , щоб встановити цей стовпець. 
    2.  Використовуйте {{ currentUserInfo.language }} , щоб встановити використання мови в ILLA.

    ```jsx
    {{currentUserInfo.language=='ja-JP' ? item.options_jp 
    : currentUserInfo.language == 'ko-KR' ? item.options_kr 
    : currentUserInfo.language == 'zh-CN' ? item.options_zh 
    : item.options_en }}
    ```
