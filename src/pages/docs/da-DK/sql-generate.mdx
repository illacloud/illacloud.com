<Heading hidden>SQL Generation</Heading>

Vi understøtter SQL generation i disse ressourcer: PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB og ClickHouse. Du kan indtaste det naturlige sprog og få en klar-til-køre SQL erklæring, når du bruger disse ressourcer. 

**Indtast appens redigeringsside og opret en handling ved hjælp af en af ressourcerne ovenfor.**
