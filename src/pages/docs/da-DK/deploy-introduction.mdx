## <Heading hidden>Implementering introduktion</Heading>

### Sky

Prøv et enkelt klik implementering i ILLA Cloud for den nemmeste oplevelse.

### Selv-vært

Du kan selv vært lokalt med ILLA CLI, Docker komponere, k8s roret osv. Vi har gentagne gange optimeret selvværtsprocessen for at gøre det muligt for dig at foretage selvværts- og softwareimplementeringer på få minutter. Sammenlignet med skyen, kan du bruge ILLA produkter mere sikkert og pålideligt, og du kan bruge ILLA Builder ved blot at besøge installationsstedet efter hosting og implementering Builder lokalt.

For mere information om selvvært, se ILA CLI, Docker komponere, k8s helm boot sider.
Hvis du har problemer med din selvinstallation, bedes du kontakte os på GitHub eller business@illasoft.com.
