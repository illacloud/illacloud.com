## <Heading hidden>Kendte problemer</Heading>

### Isæt Type Problem

Kør `illa deploy --self -port=1000 -s={your-ip-address}` eller `run-official-image.sh`, returnér følgende fejl:

`Docker reagerede med status kode 400: ugyldig mount config for type "bind": bind kilde sti findes ikke: /tmp/illa-data`

For nu, monteringstypen understøtter kun `mount`, konfigurer `docker -v` som `mount` tilstand. Eller geninstaller dockeren ved at følge scripts.

[reinstall-docker-with-ubuntu.sh](https://github.com/illacloud/deploy-illa-manually/blob/main/utils/reinstall-docker-with-ubuntu.sh)

Og kør kommandoen deploy igen.

For flere problemer: https://github.com/illacloud/illa-builder/issues
