### 1.  データベース リソースを作成

リソースはデータソースへの接続を表し、ILLAアプリケーション内で問い合わせるものです。リソースはデータソースへの接続を表し、ILLAアプリケーション内で問い合わせるものです。ILLAにログインし、**リソース**タブ>**新規作成**をクリックし、接続したいデータベースを選択してください。 **Configure a modal to add your connection information**. **Configure a modal to add your connection information**.

### 2.  データベースに問い合わせ

アプリエディタで **Action List** > select resource type and choose an existing resource or create a new resource > click **create action** and update the actions to retrieve some information from a table in your database.
