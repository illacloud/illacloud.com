## <h2 hidden>Appwriteとは何ですか？</h2>

IllaでのAppwriteは、Appwriteデータベースに接続して対話できるデータベース統合です。 Appwriteは、速度、信頼性、使いやすさで知られている一般的なオープンソースのリレーショナルデータベース管理システムです。

IraでAppwrite統合を使用すると、SQLコマンドを使用してAppwriteデータベースからデータをクエリ、挿入、更新、および削除することができます。 また、カスタムSQLクエリを作成し、Ila内で実行することもできます。 これにより、Appwriteデータベースからリアルタイムデータを表示するデータ駆動型アプリケーションやダッシュボードを簡単に構築できます。

このチュートリアルでは、ILLA Builder と Appwrite を使用した Admin Panel をいくつかの簡単な手順で作成する方法について説明します。 ILLAは、社内ツールの迅速な開発と展開を可能にする開発者向けの低コードプラットフォームです。 UI コンポーネントをドラッグ&ドロップしたり、データベースや API に接続したり、JavaScript を書いたりすることで、ページを作成することができます。 Appwriteの詳細については、ウェブサイト<https://Appwrite.io/>をご覧ください。

### Appwriteを作成

Illaアカウントにログインした後、Illaにリソースを作成するには2つの方法があります。

-   リソースに作成

Illaアカウントにサインインし、ページ上部の「**リソース**」を選択し、「**新規作成**」ボタンをクリックします。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

データベース リストから **`Appwrite`** を選択します。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、 `Test Connection` をクリックします。 はいの場合は、 `Save Resources` をクリックし、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

リソースを作成すると、Ready Appwriteが次のように表示されます。

![app_ex_ready](https://cdn.illacloud.com/official-website/img/docs/app_ex_ready.png)

-   ビルダーで作成

Illa アカウントにサインインし、`App` ページで Illa Builder でプロジェクトを作成します。 をクリックし、ページの下部にある「アクションリスト」に移動します。 `new`をクリックし、データベースのリストから`Appwrite`を選択します。 次に、下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、 `Test Connection` をクリックします。 はいの場合は、 `Save Resources` をクリックし、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

### 接続設定

ここでは、Appwrite データベースに接続するための情報を提供する必要があります。

![app_config](https://cdn.illacloud.com/official-website/img/docs/app_config.png)

| プロパティー     | 必須 | 説明                                                                                            |
| ---------- | -- | --------------------------------------------------------------------------------------------- |
| 名前         | 必須 | ILLAでアクションを作成する際のリソース名です。                                                                     |
| ホスト        | 必須 | データベースの URL または IP アドレス                                                                       |
| データベースID   | 必須 | Appwrite 内で作成された各データベースに割り当てられた一意の識別子                                                         |
| プロジェクトID   | 必須 | Appwriteプロジェクト全体を表す一意の識別子                                                                     |
| API キー     | 必須 | APIリクエストをAppwriteサーバーに送信する際の認証形式となる秘密のトークン                                                    |

Appwriteで**`Host`**、**`Database ID`**、**`Project ID`**、**`API key`**を見つけるには、 最初に、プロジェクトページの左下にある `**setting**` ボタンをクリックします。

IllaのAppwrite設定の**`ホスト`**の\*\*`プロジェクト ID`と`API エンドポイント`の`プロジェクト ID`をコピーします。

![app_setting](https://cdn.illacloud.com/official-website/img/docs/app_setting.png)

次に、同じページの `View API Keys` ボタンをクリックします。 このプロジェクトの API を作成していない場合 `Create API Key` ボタンをクリックし、Appwriteで使用するデータベースフォルダのみを選択します。

![app_api_key](https://cdn.illacloud.com/official-website/img/docs/app_api_key.png)

Illa 設定の `API キー` の `API Key Secret` をコピーします。 この二つの箇所にコピーすることができます。

![app_key_secret](https://cdn.illacloud.com/official-website/img/docs/app_key_secret.png)

残っているのは、 `Database` ページで見つけることができる`Database ID`です。 これをコピーし、IllaのAppwrite設定で貼り付けます。

![app_database](https://cdn.illacloud.com/official-website/img/docs/app_database.png)

最終的な製品は、このようなものになります。

![app_final_config](https://cdn.illacloud.com/official-website/img/docs/app_final_config.png)

### Appwriteの設定

#### 概要

| メソッド名     | 説明                                                                             |
| --------- | ------------------------------------------------------------------------------ |
| 方法        | ドキュメントの作成 ドキュメントの取得 ドキュメントの更新 ドキュメントの削除 ドキュメントの一覧                              |
| コレクションID  | データベースコレクションに割り当てられた一意の識別子                                                     |
| トランス      | JavaScriptを使用して好きなスタイルにデータを変換します                                               |

#### ドキュメントを作成

コレクション内に新しいドキュメントを作成

 ** Input ** 

| プロパティー     | 必須   | 説明                                                     |
| ---------- | ---- | ------------------------------------------------------ |
| ドキュメントID   | 必須   | データベースコレクションドキュメントに割り当てられた一意の識別子                       |
| データ        | 省略可能 | 新しいドキュメントのフィールドと値を含むオブジェクト。                            |

例えば、**`name`** と **`age`** の2つのフィールドを持つコレクションに新しいドキュメントを作成します。 1 を **`document ID`** と `{{ {"Name": "David", "Age": 18}}}` を **`data`** として使用できます。

 出力 

新しく作成されたドキュメントの ID と、作成日時などの追加メタデータを含むレスポンスオブジェクト。

![app_create_code](https://cdn.illacloud.com/official-website/img/docs/app_create_code.png)

#### ドキュメントを取得

データベース内のコレクションから特定のドキュメントを取得します

 ** Input ** 

| プロパティー   | 必須 | 説明                                                    |
| -------- | -- | ----------------------------------------------------- |
| ドキュメントID | 必須 | データベースコレクションドキュメントに割り当てられた一意の識別子                      |

例えば、ドキュメントを id **`1`** で取得できます。

 出力 

リクエストされたドキュメントのデータを含むレスポンスオブジェクト

![app_get_data](https://cdn.illacloud.com/official-website/img/docs/app_get_data.png)

返されるすべての値を取得するには、`{{Appwrite1.data}}`を使用します。

![app_get_code](https://cdn.illacloud.com/official-website/img/docs/app_get_code.png)

#### ドキュメントを更新

コレクション内の既存のドキュメントのデータを更新します。

 ** Input ** 

| プロパティー     | 必須   | 説明                                                                                 |
| ---------- | ---- | ---------------------------------------------------------------------------------- |
| ドキュメントID   | 必須   | データベースコレクションドキュメントに割り当てられた一意の識別子                                                   |
| データ        | 省略可能 | ドキュメントに割り当てる更新されたフィールドと値を含むオブジェクト。                                                 |

例えば、**`updateDocument`** メソッドを使用してコレクション内のドキュメントを更新するには ドキュメント ID には **`1`** 、データには `{{{"Name":"Fred"}}}` を入れることができます。

 出力 

リクエストされたドキュメントのデータを含むレスポンスオブジェクト

![app_update)data](https://cdn.illacloud.com/official-website/img/docs/app_update_data.png)

#### ドキュメントを削除

データベース内のコレクションから特定のドキュメントを削除します

 ** Input ** 

| プロパティー   | 必須 | 説明                                                    |
| -------- | -- | ----------------------------------------------------- |
| ドキュメントID | 必須 | データベースコレクションドキュメントに割り当てられた一意の識別子                      |

例えば、id **`1`**でドキュメントを削除できます。

 出力 

文書が正常に削除されたことを確認するレスポンスオブジェクト。

![app_delete_data](https://cdn.illacloud.com/official-website/img/docs/app_delete_data.png)

#### ドキュメントの一覧

データベース内のコレクションから複数のドキュメントを取得します

 ** Input ** 

| プロパティー    | 必須   | 説明                                                                   |
| --------- | ---- | -------------------------------------------------------------------- |
| フィルター     | 省略可能 | 特定のフィールドと値に基づいてドキュメントをフィルタリングする条件を定義します。                             |
| 並び順       | 省略可能 | 結果ドキュメントをソートするフィールドを指定します。                                           |
| 制限        | 省略可能 | 結果セットに返されるドキュメントの数を制限する                                              |

例えば、Name = Jamesのあるすべてのドキュメントを一覧表示します。

 出力 

フィルタを満たす文書の配列。

![app_list_data](https://cdn.illacloud.com/official-website/img/official-site/intergration/app_list_data.png)

返された値を取得するには、`{{Appwrite1.data[0].documents}}`を使用します。

![app_list_code](https://cdn.illacloud.com/official-website/img/docs/app_list_code.png)
