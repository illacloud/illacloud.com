## <h2 hidden>ClickHouseとは何ですか？</h2>

ClickHouseは、列指向の高速データベース管理システムで、オンライン分析処理(OLAP)ワークロード用に設計されています。 これは、ロシアの検索エンジン会社であるYandexによって作成されたオープンソースプロジェクトです。 ClickHouseは大量のデータを迅速かつ効率的に処理できることで知られており、ビッグデータ分析に理想的なソリューションです。

Illa Cloudを使用すると、基盤となるインフラストラクチャとメンテナンスを心配することなく、ClickHouseクラスタをクラウド上に展開および管理できます。

Illa CloudはClickHouseクラスタを管理するためのWebベースのインターフェースを提供します。 クラスタのパフォーマンスを監視し、リソースを上下にスケールアップし、バックアップとセキュリティ設定を構成できます。 Illa Cloud を使用して、Web ベースのクエリ エディタを使用して ClickHouse データのクエリを実行することもできます。

### ClickHouseを作成

Illaアカウントにログインした後、Illaにリソースを作成するには2つの方法があります。

-   リソースに作成

Illaアカウントにサインインし、ページ上部の「**リソース**」を選択し、「**新規作成**」ボタンをクリックします。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

データベースリストから **`ClickHouse`** を選択します。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

リソースを作成すると、Ready ClickHouseが表示されます。

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

-   ビルダーで作成

Illa アカウントにサインインし、`**App**` ページで Illa Builder でプロジェクトを作成します。 をクリックし、ページの下部にある **`Action List`** に移動します。 `**new**`をクリックし、データベースリストから**\``ClickHouse`**を選択します。 次に、下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

### 接続設定

ここでは、ClickHouse データベースに接続するための情報を提供する必要があります。

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

| プロパティー      | 必須   | 説明                                                                                                        |
| ----------- | ---- | --------------------------------------------------------------------------------------------------------- |
| 名前          | 必須   | ILLAでアクションを作成する際のリソース名です。                                                                                 |
| Hostname    | 必須   | データベースの URL または IP アドレス                                                                                   |
| ポート         | 必須   | 接続に使用するサーバーホストのポート番号。 ポートを指定しない場合、デフォルトのポートは '9440' です。                                                   |
| データベース      | 必須   | データベースの名前                                                                                                 |
| ユーザー名       | 必須   | ClickHouseサーバーへのログイン時に使用するユーザー名。                                                                          |
| パスワード       | 必須   | 認証にこのパスワードを使用してください。                                                                                      |
| SSL オプション   | 省略可能 | サーバとのネゴシエーション中にセキュアな SSL TCP/IP 接続が優先される方法を決定します。                                                         |

### アクションを作成

ClickHouseリソースを作成しました。アクションリストからClickHouseを選択し、**`アクションを作成`**ボタンを選択することでアクションを追加できます。

![click_resource_list](https://cdn.illacloud.com/official-website/img/docs/clik_resource_list.png)

これでClickHouseサーバーがビルディングページにアクションとして追加されました。

### Configure ClickHouse

| プロパティー    | 説明                                                               |
| --------- | ---------------------------------------------------------------- |
| SQL クエリ   | データベースからデータを取得(選択)、挿入、更新、削除するための sql コマンド                        |
| トランス      | JavaScriptを使用して好きなスタイルにデータを変換します                                 |

使用例:

-   選択

```sql
SELECT * FROM users ORDER BY last name;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   更新

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   削除

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```
