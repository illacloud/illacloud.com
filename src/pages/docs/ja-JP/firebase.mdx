## <h2 hidden>Firebaseとは？</h2>

Firebaseは、Googleが提供する包括的なプラットフォームで、Webやモバイルアプリケーションの開発や管理のための幅広いツールやサービスを提供しています。 開発者にスケーラブルで安全なバックエンドインフラストラクチャを提供し、開発プロセスを加速するための事前構築された機能と機能のスイートを提供します。 Firebase を ILLA Cloud に統合することで、開発者は信頼性が高くスケーラブルなストレージソリューションを使用してアプリケーションを強化できます。 革新的でデータ駆動型のエクスペリエンスの構築に集中できるようになります

### Firebase API の作成

Illaアカウントにログインした後、Illaにリソースを作成するには2つの方法があります。

-   リソースに作成

Illaアカウントにサインインし、ページの上部にある`**リソース**`を選択し、`**新規作成**`ボタンをクリックします。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

API リストから **`Firebase`** を選択します。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

リソースを作成すると、次のように準備完了のFirebaseが表示されます。

![firebase_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/firebase_ex_ready.jpg)

-   ビルダーで作成

Illa アカウントにサインインし、`**App**` ページで Illa Builder でプロジェクトを作成します。 をクリックし、ページの下部にある **`Action List`** に移動します。 `**new**`をクリックし、データベースリストから**`Firebase`**を選択します。 次に、下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

#### UIDでユーザーを取得

ユーザー固有識別子(UID)を指定してユーザー情報を取得します。 これにより、ユーザーに関連付けられた表示名、電子メール、電話番号、カスタム属性などのユーザー情報を取得できます。

** Input **

| プロパティー     | 必須 | 説明                                                                                               |
| ---------- | -- | ------------------------------------------------------------------------------------------------ |
| UID        | 必須 | 「ユーザー識別子」 これは、Firebase 認証システム内の各ユーザーに割り当てられた一意の識別子です。                                            |

#### メールでユーザーを取得

ユーザのメールアドレスを指定してユーザ情報を取得します。 これにより、UIDベースのメソッドと同様のユーザー詳細を取得できますが、電子メールを識別子として使用できます。

** Input **

| プロパティー       | 必須 | 説明                                                        |
| ------------ | -- | --------------------------------------------------------- |
| メールアドレス      | 必須 | 取得したいユーザーに関連付けられた登録されたメール                                 |

#### 電話番号でユーザーを取得
