## <Heading hidden>Supabase DB</Heading>

Supabaseは、開発者がリアルタイムでWebやモバイルアプリケーションを構築できるオープンソースのFirebaseの代替手段です。 リアルタイムデータベースやユーザー認証など、Firebaseと同じ多くの機能を提供します。 しかし、オープンソースであり、SQL APIを提供するという利点があります。 これにより、開発者は既存のSQL知識とツールを使用できるため、他のシステムと簡単に統合できます。 さらに、SupabaseはFirebaseよりも柔軟でカスタマイズ可能なプラットフォームを提供します。このプラットフォームは、データモデリングや使用についてより制限をかけることができます。

このチュートリアルでは、ILLA BuilderとSupabaseを使用したAdmin Panelを簡単な手順で作成する方法について説明します。 ILLAは、社内ツールの迅速な開発と展開を可能にする開発者向けの低コードプラットフォームです。 UI コンポーネントをドラッグ&ドロップしたり、データベースや API に接続したり、JavaScript を書いたりすることで、ページを作成することができます。 Supabaseの詳細については、ウェブサイト<https://supabase.com/>をご覧ください。 始めましょう！

### ステップ3：Supabaseに接続してコンフィグCRUD

Supabase の [Project Settings](https://app.supabase.com/project/hdcfnsxpwwgboqomdrhp/settings/database) のデータベース接続情報に注意してください。

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase6.png'
  alt=""
/>

アクションリストで`+ New`をクリックし、Supabase DBを選択します。

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase7.png'
  alt=""
/>

フォームに記入して、Supabaseインスタンスに接続します。 接続をテストしてリソースを保存します。

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase8.png'
  alt=""
/>

`Create Action`をクリックして、Supabaseリソースでアクションを作成し、CRUDを設定します。

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase9.png'
  alt=""
/>

フロントエンドの入力データを取得するには {{ を使用します。 以下は、format@@0パネルのformat@@1ページの例です。

input1 に入力された名前でユーザーを検索

```
SELECT *
FROM user
WHERE名 = "{{input1.value}}"
;
```

ユーザーデータを更新 IDが一致したときにユーザー情報を更新する

```
UPDATE user
SET name = "{{input3.value}}"
, email = "{{input4.value}}"
WHERE id="{{input2.value}}"
;
```

ユーザーデータを挿入

```
INSERT INTO user VALUES("{{input5.value}}",",{{input6.value}}",",{{input7.value}}');
```

IDでユーザーを削除する

```
ユーザーWHEREから削除 ID = "{{input2.value}}";
```
