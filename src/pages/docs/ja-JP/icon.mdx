## <h2 hidden>アイコンとは</h2>

ILLA Cloud の Icon コンポーネントは、ユーザーがアプリケーションの視覚的な魅力と機能を強化することができる強力な機能です。 Icon コンポーネントを使用すると、ユーザーは、人気のアイコンライブラリからユーザーインターフェイスに簡単にアイコンを追加できます。 より直感的で魅力的な体験をユーザーに提供します Icon コンポーネントを使用すると、ユーザーは、人気のアイコンライブラリからユーザーインターフェイスに簡単にアイコンを追加できます。 より直感的で魅力的な体験をユーザーに提供します

### プロパティー

| プロパティー        | 説明                                                                                                                         |
| ------------- | -------------------------------------------------------------------------------------------------------------------------- |
| アイコン          | ユーザーはあらかじめ定義されたオプションからアイコンを簡単に選択できます                                                                                       |
| イベントハンドラー     | トリガクエリ、コンポーネントのイベントに応じてコンポーネントを制御したり、他のAPIを呼び出したりします。                                                                      |
| ツールチップ        | ユーザはコンポーネントツールチップを入力できます。 ツールチップがフォーカスされると表示されます。 Markdown 形式がサポートされています。 ツールチップがフォーカスされると表示されます。 Markdown 形式がサポートされています。  |
| Hidden        | コンポーネントが非表示になるかどうかを動的に制御します。 非表示状態は動的ブール値で変更できます。 非表示状態は動的ブール値で変更できます。                                                     |
| テーマの色         | ボタンの背景色と不透明度を指定できます                                                                                                        |

### イベントハンドラ：

| イベント  | 説明                                                               |
| ----- | ---------------------------------------------------------------- |
| Click | ユーザーがこのアイコンをクリックすると、ユーザーがカスタマイズした特定のアクションを実行します。                 |

### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名        | 説明                                                                                 |
| ------------- | ---------------------------------------------------------------------------------- |
| colorScheme   | アイコンのストローク色                                                                        |
| displayName   | このコンポーネントの名前 (つまりアイコン1)                                                            |
| イベント          | コンポーネント内で特定のアクションやイベントが発生したときに生成またはトリガーされます。                                       |
| hidden        | hidden status (true または false)                                                     |
| iconName      | アイコンのラベル名                                                                          |
| tooltipText   | ツールチップテキストの値                                                                       |

例: {{icon1.iconName}}
