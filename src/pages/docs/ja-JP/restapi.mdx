| URL パラメータ     | 省略可能 | 追加情報を提供したり、API リクエストの動作を変更します。                                                |
| ヘッダー          | 省略可能 | API リクエストまたはレスポンスに関する追加メタデータ                                                  |
| Cookie        | 省略可能 | クライアント側に保存され、APIリクエストごとに送信される小さなデータ                                           |
| 認証            | 省略可能 | APIリクエストを行うユーザーまたはアプリケーションの身元を確認する                                            |
