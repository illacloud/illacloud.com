## アセンブルコンポーネント

アプリのユーザーインターフェイスを構築するために、ILLAのUIコンポーネントのライブラリを使用してください。

コンポーネントは ILLA の必須要素であり、プロパティを受け入れて内部の state を持つことができるという点で React コンポーネントと同様に機能します。 ILLAのコンポーネントライブラリとデータを組み合わせることで、インタラクティブなアプリを構築できます。 ILLAのコンポーネントライブラリとデータを組み合わせることで、インタラクティブなアプリを構築できます。

いくつかの一般的なコンポーネント:

-   テーブルコンポーネントは表形式でデータを表示するために使用され、ソート、フィルタリング、ページネーション、CSV エクスポートなどの機能を提供します。
-   フォームコンポーネントは、入力フィールドをグループ化し、単一の単位として送信するために使用されます。
-   Select 入力コンポーネントはドロップダウンと入力フィールドの組み合わせで、ユーザーはリストから値を選択するか、テキストとして直接値を入力することができます。
-   チャートコンポーネントは、グラフとグラフを使用してデータの可視化を作成するために使用されます。
