## <h2 hidden>Τι είναι το Χρονοδιάγραμμα?</h2>

Ένα στοιχείο χρονοδιαγράμματος είναι μια γραφική αναπαράσταση γεγονότων ή σημείων δεδομένων τοποθετημένα χρονολογικά κατά μήκος ενός οριζόντιου ή κατακόρυφου άξονα. Επιτρέπει στους χρήστες να οπτικοποιήσουν και να περιηγηθούν σε μια σειρά από γεγονότα ή ορόσημα με γραμμικό τρόπο.

## \***\*Ιδιότητες\*\***

Διαθέσιμες ιδιότητες του Timeline. Η JavaScript μπορεί να γραφτεί για να διαβάσει ή να τροποποιήσει πληροφορίες σχετικά με τα στοιχεία.

| Ιδιότητες     | Περιγραφή                                                                                              |
| ------------- | ------------------------------------------------------------------------------------------------------ |
| Στοιχεία      | τη συλλογή μεμονωμένων συμβάντων ή σημείων δεδομένων που εμφανίζονται στο χρονοδιάγραμμα               |
| Κατεύθυνση    | τον προσανατολισμό ή τη διάταξη του συστατικού του χρονοδιαγράμματος (οριζόντια ή κατακόρυφα)          |
| Εκκρεμεί      | την κατάσταση ενός στοιχείου στο χρονοδιάγραμμα                                                        |
| Ύψος          | κατακόρυφο μέγεθος ή ύψος του στοιχείου χρονοδιαγράμματος                                              |
| Hidden        | αν η συνιστώσα του χρονοδιαγράμματος είναι ορατή ή κρυμμένη                                            |

## \***\*Μέθοδοι\*\***

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες δύο μεθόδους:

-   **ορισμός**

Για να ορίσετε την τιμή εισόδου της περιοχής κειμένου, για παράδειγμα, {{"value1"}}

| Ιδιότητες | Περιγραφή   |
| --------- | ----------- |
| Τιμή      | Input value |

-   **καθαρή αξία**

Για να καθαρίσετε την τιμή του επιλεγμένου στοιχείου

## Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας          | Περιγραφή                                                                                                                      |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| κατεύθυνση               | προσανατολισμός ή διάταξη του χρονοδιαγράμματος                                                                                |
| δυναμικόΎψος             | την ικανότητα δυναμικής προσαρμογής του ύψους του χρονοδιαγράμματος με βάση το περιεχόμενό του ή άλλους παράγοντες.            |
| Εμφάνιση Ονόματος        | το όνομα αυτού του στοιχείου (δηλαδή χρονοδιαγράμματος1)                                                                       |
| hidden                   | αν η συνιστώσα του χρονοδιαγράμματος είναι ορατή ή κρυμμένη                                                                    |
| αντικείμενα              | τη συλλογή μεμονωμένων συμβάντων ή σημείων δεδομένων που εμφανίζονται στο χρονοδιάγραμμα                                       |
| εκκρεμεί                 | την κατάσταση ενός στοιχείου στο χρονοδιάγραμμα                                                                                |
| αλλαγή μεγέθουΚατεύθυνση | την κατεύθυνση κατά την οποία το στοιχείο χρονοδιαγράμματος μπορεί να αλλάξει μέγεθος από το χρήστη                            |

## Περίπτωση χρήσης

Στη συνέχεια, θα καταδείξουμε πώς να χαρτογραφήσουμε τα δεδομένα από την πηγή δεδομένων στο στοιχείο χρονοδιαγράμματος.

#### Βήμα 1 Προσθέστε μια ενέργεια

Ας δημιουργήσουμε έναν πίνακα στο Supabase που ονομάζεται **`timeline`,** που περιλαμβάνει μόνο 1 στήλες: ορόσημα. Υπάρχουν 4 ορόσημα που θέλουμε να χαρτογραφήσουμε με το συστατικό χρονοδιάγραμμα .

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα δράση για το Supabase από τη λίστα ενεργειών και το όνομα supabasedb1.

Για να απαριθμήσετε όλα τα δεδομένα σε ορόσημο, βάλτε το απόσπασμα κώδικα παρακάτω στην ενότητα ερώτημα SQL.

```sql
επιλέξτε * από το χρονοδιάγραμμα;
```

Δεδομένου ότι τα δεδομένα εξόδου είναι μια λίστα με ζεύγος λεξικών, θέλουμε να τα μετατρέψουμε σε ένα πίνακα. Στο Transformer, ενεργοποιήστε το και βάλτε τον παρακάτω κώδικα.

```jsx
const transformedData = data.map((obj) => Object.values(obj)[0]);
return transformedData.
```

Πατήστε **`Save`** και **`Run`** για να ενεργοποιήσετε αυτήν την ενέργεια.

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### Βήμα 2 Προσθήκη Συστατικών

Στη συνέχεια, μπορούμε να προσθέσουμε ένα στοιχείο **`timeline`** στον καμβά.

Στο πεδίο `**Αντικείμενα**` στη σελίδα Επιθεώρηση αφού κάνετε κλικ στο στοιχείο, βάλτε το `{{supabasedb1.data}}` για να ανακτήσετε τα δεδομένα από supabase.

Το τελικό βλέμμα θα πρέπει να είναι όπως φαίνεται.
