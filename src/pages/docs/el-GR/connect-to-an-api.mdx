## <Heading hidden>Συνδεθείτε σε ένα API</Heading>

Συνδεθείτε σε οποιοδήποτε REST, GraphQL API.

Μπορείτε να συνδέσετε το ILLA με τα περισσότερα APIs χρησιμοποιώντας τις εγγενείς ενοποιήσεις μας. Αν χρειαστεί να συνδεθείτε με ένα εσωτερικό API, ή αν δεν υπάρχει υποστηριζόμενη ενσωμάτωση, μπορείτε να συνδεθείτε χρησιμοποιώντας την ολοκλήρωση REST API ή την ολοκλήρωση GraphQL.

### 1.  Δημιουργία ενός πόρου API

Οι πόροι αντιπροσωπεύουν συνδέσεις με πηγές δεδομένων και είναι αυτό που ρωτάτε στις εφαρμογές ILLA.Συνδεθείτε στο ILLA και μεταβείτε στην καρτέλα **Resources** > κάντε κλικ στο **Δημιουργία νέου** και επιλέξτε το API στο οποίο θέλετε να συνδεθείτε, όπως **REST API**. Ανοίγει τη λειτουργία **Ρυθμίστε μια λειτουργία πόρου** όπου προσθέτετε τις πληροφορίες σύνδεσης.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

Στο ελάχιστο, χρειάζεστε τη βασική διεύθυνση URL του API που θέλετε να συνδεθείτε. Μπορείτε επίσης να περάσετε παραμέτρους URL, κεφαλίδες, κλπ. Αν το API σας απαιτεί έλεγχο ταυτότητας, μπορείτε να το ρυθμίσετε και σε αυτή τη σελίδα. Για παράδειγμα, το OAuth2.0 απαιτεί URL εξουσιοδότησης, URL διακριτικού πρόσβασης, κλπ. Αφού εισάγετε τα στοιχεία σύνδεσής σας, κάντε κλικ στο κουμπί **Αποθήκευση πόρου.**

### 2.  Ερώτημα API

Στον επεξεργαστή εφαρμογών, κάντε κλικ στο **+ Νέο** στη **Λίστα ενεργειών** > επιλέξτε **REST API** και επιλέξτε ένα υπάρχον API REST ή δημιουργήστε ένα νέο> κάντε κλικ στο **Δημιουργία ενέργειας** και ενημερώστε τις ενέργειες με πρόσθετες παραμέτρους URL, κεφαλίδες κ.λπ.
