## <h2 hidden>Τι είναι Ημερομηνία εύρος?</h2>

Εφαρμογή Εύρος ημερομηνιών είναι ένα στοιχείο UI που επιτρέπει στους χρήστες να επιλέξουν ένα εύρος ημερομηνιών από μια διεπαφή ημερολογίου, χρησιμοποιείται συνήθως σε μορφές ή διεπαφές εισόδου δεδομένων όπου ο χρήστης πρέπει να καθορίζει ένα εύρος ημερομηνιών έναρξης και λήξης.

Όταν ένα στοιχείο Εύρος ημερομηνιών προστίθεται σε καμβά, εμφανίζει δύο διεπαφές ημερολογίου δίπλα-δίπλα. Μια διεπαφή ημερολογίου χρησιμοποιείται για την επιλογή της ημερομηνίας έναρξης, ενώ η άλλη χρησιμοποιείται για την επιλογή της ημερομηνίας λήξης. Ο χρήστης μπορεί να περιηγηθεί στα ημερολόγια κάνοντας κλικ στα προηγούμενα ή τα επόμενα βέλη για να μετακινηθεί σε διαφορετικό μήνα, ή κάνοντας κλικ σε μια συγκεκριμένη ημερομηνία για να την επιλέξετε. Οι επιλεγμένες ημερομηνίες θα εμφανίζονται στα πεδία εισαγωγής.

### Properties

Εύρος ημερομηνιών διαθέσιμες ιδιότητες. Η JavaScript μπορεί να γραφτεί για να διαβάσει ή να τροποποιήσει πληροφορίες σχετικά με τα στοιχεία.

| Μέγιστη ημερομηνία            | ορισμός της μέγιστης ημερομηνίας που ένας χρήστης μπορεί να επιλέξει από τη διεπαφή ημερολογίου.                                                                                 |

| Θέση                          | Ορίστε τη θέση της ετικέτας σε σχέση με το στοιχείο                                                                                                                              |

| Μόνο Για Ανάγνωση             | Ελέγξτε την κατάσταση του αν το στοιχείο είναι μόνο για ανάγνωση. Ένα στοιχείο μόνο για ανάγνωση μπορεί να επιλεγεί και να εστιασθεί, αλλά δεν μπορεί να τροποποιηθεί.           |

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να χαρτογραφήσουμε τα δεδομένα από την πηγή δεδομένων έως το εύρος ημερομηνιών και να ορίσουμε την τελική τιμή μέχρι τη σημερινή ημερομηνία με ένα κουμπί. 

#### Βήμα 1 Προσθέστε μια ενέργεια

Επιτρέψτε μας να δημιουργήσουμε έναν πίνακα στο REST API που ονομάζεται **`promoteCodes`,** συμπεριλαμβανομένων 6 στηλών: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. Θα χρησιμοποιήσουμε το αρχικό και έληξεΚατά το χρόνο για τη χρήση δειγμάτων. 

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα δράση για REST API από τη λίστα δράσης, λίστα όλων των δεδομένων σε promoteCodes και ονομάζεται restapi1. 

Πατήστε **`Save`** και **`Run`** για να ενεργοποιήσετε αυτήν την ενέργεια.

#### Βήμα 2 Προσθήκη Συστατικών

Στη συνέχεια, μπορούμε να προσθέσουμε ένα **`date range`** component και ένα **`button`** component στον καμβά. Για την προεπιλεγμένη τιμή ημερομηνίας έναρξης του στοιχείου εύρος ημερομηνιών, χρησιμοποιούμε την εκκίνησηΚατά την ώρα της πρώτης προώθησης κώδικα από τα δεδομένα υπόλοιπου api.

```sql
{{restapi1.data.promoteCodes[0].startedAt}}
```

Ομοίως, για την προεπιλεγμένη τιμή ημερομηνίας λήξης, χρησιμοποιούμε έληξεΚατά τη στιγμή της πρώτης προώθησης κώδικα από τα δεδομένα υπόλοιπο api.

```sql
{{restapi1.data.promoteCodes[0].expiredAt}}
```

Σημειώσαμε επίσης το κουμπί ως 'Ενημέρωση', όπως φαίνεται παρακάτω.

![date_range_0](https://cdn.illacloud.com/official-website/img/docs/date_range_0.png)

#### Βήμα 3 Διαμόρφωση του εξαρτήματος

Για το **`button`** component, μπορούμε να το ρυθμίσουμε ώστε να ορίζει την τιμή λήξης σε **`date range`** component στη σημερινή ημερομηνία.

1.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, επιλέξτε το **`date range`** component που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
2.  Επιλέξτε την **`setEndValue`** ενέργεια και ορίστε την **`Ημερομηνία λήξης`** ως `{{currentUserInfo.updatedAt}}` 

![date_range_config](https://cdn.illacloud.com/official-website/img/docs/date_range_config.png)

#### 4Ο Στάδιο Δοκιμή

Τώρα που όταν κάνετε κλικ στο κουμπί 'Ενημέρωση', η ημερομηνία λήξης θα πρέπει να αλλάξει τη σημερινή ημερομηνία. Όσο για εδώ, έχουμε 2023-5-6 αλλά μπορεί να είναι διαφορετικό όταν το δοκιμάσετε.
