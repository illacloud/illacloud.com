## <h2 hidden>Προσαρμοσμένη πιστοποίηση API</h2>

Το ILLA υποστηρίζει από προεπιλογή πολλά πρότυπα ελέγχου ταυτότητας (δείτε εδώ για περισσότερες πληροφορίες). 
Επιπλέον, η ILLA έχει μια προσαρμοσμένη λειτουργία ελέγχου ταυτότητας που σας επιτρέπει να συνδυάσετε πολλαπλά βήματα υποστηρίζοντας σχεδόν οποιαδήποτε πιστοποίηση API. 
Αυτές οι μέθοδοι ελέγχου ταυτότητας μπορούν να χρησιμοποιηθούν τόσο για τους πόρους REST όσο και για τους πόρους GraphQL.
