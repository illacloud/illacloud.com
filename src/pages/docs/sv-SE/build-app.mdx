Med vårt omfattande bibliotek av UI-komponenter kan du snabbt och enkelt skapa mycket polerade gränssnitt. Du kan skriva frågor för att interagera med data och använda JavaScript i många olika delar av din applikation.
Redaktör
Skapa, distribuera och köra appar med ILLA:s webbaserade utvecklingsmiljö 
Design och användbarhet (Bilder, Transformatorer)
Anpassa appar med anpassade stilar och formatering
Händelsehanterare 
Händelsehanterare låter dig utlösa frågor, kontrollera komponenter och utföra andra åtgärder som svar på användarinteraktioner
Montera komponenter
Använd ILLA:s bibliotek med UI-komponenter för att bygga appens användargränssnitt.
Komponenter är en viktig del av ILLA, och de fungerar på samma sätt som React komponenter genom att de kan acceptera egenskaper och ha internt tillstånd. Genom att kombinera ILLA:s komponentbibliotek med dina data kan du bygga interaktiva appar.
Några vanliga komponenter inkluderar:
Tabellkomponenten används för att visa data i tabellformat, och det erbjuder funktioner som sortering, filtrering, sidnumrering och CSV-export. 
Formulärkomponenten används för att gruppera inmatningsfält och skicka dem som en enda enhet. 
Den valda inmatningskomponenten är en kombination av en rullgardinsmeny och ett inmatningsfält som tillåter användare att antingen välja ett värde från en lista eller ange ett värde direkt som text. 
Diagramkomponenten används för att skapa visualiseringar av data med hjälp av diagram och grafer.
