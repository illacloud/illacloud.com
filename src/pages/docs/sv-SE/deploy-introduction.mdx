## <Heading hidden>Distribuera introduktion</Heading>

### Moln

Prova ett klick distribution i ILLA Cloud för den enklaste upplevelsen.

### Själv-värd

Du kan själv vara värd lokalt med ILLA CLI, Docker komponera, k8s rodret etc. Vi har upprepade gånger optimerat processen för att göra det möjligt för dig att göra själv-värd- och programvaruinstallationer på bara några minuter. Jämfört med molnet kan du använda ILLA produkter säkrare och tillförlitligare. och du kan använda ILLA Builder genom att helt enkelt besöka distributionssajten efter hosting och distribution av Builder lokalt.

För mer information om självvärd, se ILA CLI, Docker compose, k8s romsstartsidor.
Om du har problem med din egen driftsättning, vänligen kontakta oss på GitHub eller business@illasoft.com.
