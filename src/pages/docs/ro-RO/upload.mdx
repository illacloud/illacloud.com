## <Heading hidden>Încărcați</Heading>

Componenta de încărcare este utilizată pentru a permite utilizatorilor să încarce fișiere de pe aparatele lor locale în orice spațiu de stocare din cloud prin API. Apoi, vom introduce cum să configurăm o componentă de încărcare, cum să asociem componenta de încărcare cu un serviciu de stocare cloud, cum să utilizaţi componenta de încărcare pentru a selecta o imagine, şi să o afişaţi cu o imagine, etc.

### Metodă

Puteți folosi alte componente pentru a controla componenta Încărcare. Sprijinim următoarele trei metode:

| clearValidare | Folosit pentru a șterge mesajul de validare                             |
| ------------- | ----------------------------------------------------------------------- |
| curățare      | Folosit pentru a şterge selecţia                                        |
| Dezactivat    | Folosit pentru a seta componenta de încărcare la dezactivat sau activat |

### Gestionar de evenimente

Pentru a declanșa acțiuni atunci când se schimbă fișierele selectate

### Datele încărcătorului

Componenta de încărcare are niște date utilizate în mod obișnuit, care pot fi apelate în aplicație.

`value` un array format din datele base64 ale fișierelor. 

`files` o serie de obiecte. Cheile obiectelor includ `lastModified`, `name`, `size`, `type`. Puteți utiliza `{{upload1.files.map(fișier =>fișier). ame)}` pentru a obţine o serie de nume de fişiere şi puteţi obţine de asemenea matrice de ultimă oră, matrice de mărime, matrice în acelaşi mod. 

### Încărcați fișiere într-un serviciu de stocare în cloud

<img src='https://cdn.illacloud.com/official-website/img/docs/upload2.gif' alt="" />

#### Încărcați un singur fișier printr-o resursă S3

Configurația acțiunii este următoarea. 

Încărcați numele obiectului

```jsx
{{upload1.files.map(fișier =>fișier.name)[0]}}
```

Încarcă date

```jsx
{{upload1.value[0]}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload3.png' alt="" />

#### Încarcă mai multe fișiere printr-o resursă S3

Încărcați lista de nume de obiecte

```jsx
{{upload1.files.map(fișier =>file.name)}}
```

Încarcă lista de date

```jsx
{{upload1.value}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload4.png' alt="" />

Încărcaţi o imagine şi afişaţi-o în componenta de imagine
Modificaţi sursa imaginii în baza de fişier64 a componentei de încărcare. Datele base64 sunt stocate în atributul valoric din array. Și ar trebui să împărțim prefixul fișierului cu datele fișierului. Codul este după cum urmează. 

```jsx
{{'data:image/jpeg;base64,'+upload1.value[0]}}
```
