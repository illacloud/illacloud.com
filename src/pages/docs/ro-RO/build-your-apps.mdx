## Componente de asamblare

Folosește biblioteca ILLA a componentelor UI pentru a construi interfața de utilizator a aplicației tale.

Componentele sunt o parte esențială a ILLA, și funcționează similar cu componentele React prin faptul că pot accepta proprietăți și pot avea stare internă. Prin combinarea bibliotecii de componente ILLA cu datele dvs., puteți construi aplicații interactive.

Unele componente frecvente includ:

-   Componenta Tabel este utilizată pentru a afișa date într-un format tabular și oferă caracteristici precum sortare, filtrare, paginare și exporturi CSV.
-   Componenta formular este folosită pentru a grupa câmpurile de intrare și a le trimite ca o singură unitate.
-   Selectaţi componenta de intrare este o combinaţie între un dropdown şi un câmp de intrare care permite utilizatorilor fie să selecteze o valoare dintr-o listă, fie să introducă o valoare direct ca text.
-   Componenta grafică este utilizată pentru a crea vizualizări de date folosind grafice și grafice.
