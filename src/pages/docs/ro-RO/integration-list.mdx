## <Heading hidden>Lista de integrare</Heading>

ILLA oferă o gamă de integrări pre-construite care vă permit să vă conectați cu ușurință la sursele de date cu o configurație minimă. Pur şi simplu selectaţi integrarea adecvată din lista de opţiuni disponibile, introduceţi acreditările necesare şi puteţi începe să utilizaţi sursa de date în cadrul ILLA. Acest lucru elimină necesitatea de a configura configurații complexe și vă permite să începeți să lucrați rapid cu datele dvs.

### Integrările bazei de date

| Nume             | Descriere                                                                                                                                                                                                                                               |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql            | MySQL este un sistem open-source de management relațional al bazei de date (RDBMS) bazat pe limbajul interogării structurate (SQL).                                                                                                                     |
| PostgreSQL       | PostgreSQL este un sistem open-source de management al bazelor de date relaționale cu obiectele (ORDBMS) cunoscut pentru fiabilitatea și bogăția sa.                                                                                                    |
| MariaDB          | MariaDB este un sistem open-source de management al bazei de date relaționale, care este o furcă a bazei de date MySQL.                                                                                                                                 |
| TiDB             | TiDB este o bază de date NewSQL open-source care suportă sarcinile de procesare transacțională și analitică Hybrid.                                                                                                                                     |
| Redis            | Redis este un stoc open-source de date în memorie care este adesea folosit ca un agent de date, cache și mesaje.                                                                                                                                        |
| MongoDB          | MongoDB este un program gratuit și open-source de baze de date bazat pe documente.                                                                                                                                                                      |
| Căutare elastică | Elasticsearch este un motor de căutare open-source, distribuit si analytics construit pe Apache Lucene.                                                                                                                                                 |
| Firebase         | Firebase este o platformă de dezvoltare a aplicațiilor mobile și web bazată pe cloud dezvoltată de Google.                                                                                                                                              |
| Supabase DB      | Supabase este o platformă open-source pentru construirea de aplicații serverless în timp real. Aceasta este construită pe PostgreSQL și oferă o serie de instrumente și servicii pentru construirea și implementarea rapidă a aplicațiilor fără server. |
| Clickhouse       | ClickHouse este un sistem open-source de gestionare a bazei de date orientat către coloane, conceput pentru procesarea analitică online (OLAP) și pentru analiza datelor în timp real.                                                                  |

### Integrări API

| Nume                  | Descriere                                                                                                                                       |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| API-ul REST           | REST este un stil arhitectural pentru construirea de servicii web. API RESTful sunt API-uri care urmează stilul arhitectural al REST.           |
| Amazon S3(General S3) | Amazon S3 este un serviciu de stocare a obiectelor oferit de AWS. Acesta oferă o stocare sigură, scalabilă și ieftină a datelor de pe internet. |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) este un protocol pentru trimiterea şi primirea mesajelor de e-mail pe internet.                            |
| GraphQL               | GraphQL este o limbă de interogare pentru API-uri și o execuție pentru îndeplinirea acelor interogări cu datele existente.                      |
