## <Heading hidden>Editor de aplicaciones</Heading>

El editor de APP es la interfaz para desarrollar la aplicación y consta de las siguientes partes principales:

-   Barra de herramientas
-   Frame
-   Editor de acciones
-   Área de datos
-   Insert
-   Inspeccionar

<img alt="editor" src={'https://cdn.illacloud.com/official-website/img/official-site/content.png'} />

### Barra de herramientas

La barra de herramientas se encuentra en la parte superior del Editor de APP, que se utiliza principalmente para configurar la APP y se puede utilizar para las siguientes tareas：

-   Haga clic en el icono de ILLA a la izquierda para volver a la página de inicio.
-   Ver información de la APP y guardar estado.
-   Mostrar / ocultar elementos izquierdo, derecho, panel inferior.
-   Ajustando la escala de objetos.
-   Mostrar todos los desarrolladores actuales de la APP.
-   Alertando la APP al estado actual de error.
-   Comparte la APP con otros desarrolladores.
-   Ejecutando la APP.

#### Más características vienen soom

-   Importación de API de json.
-   APP de reproducción.
-   Exportar API en formato json.
-   Seleccione la versión de APP deseada.
-   Ver el historial editado.
-   Recargar todas las acciones.

### Frame

Frame es el área de la APP donde se puede editar la interfaz gráfica de la APP. Puede arrastrar y soltar insertos en el marco desde el panel de inserciones de la mano derecha; también puede seleccionar las inserciones que se han colocado en el marco y utilizar el ratón para arrastrar y soltar para modificar la posición y el tamaño de las inserciones.

Cuando arrastras y sueltas inserciones o modificas su tamaño, un cuadrado de guiones aparecerá en el fondo del marco para ayudarle a organizar las inserciones de forma clara en el marco. Cuando encuentras inserciones superpuestas mientras arrastra y solta, ILLA Builder dará prioridad a la inserción que está moviendo y automáticamente moverá la inserción de bloqueo fuera de su camino.

#### Vista previa

El botón del interruptor de vista previa se encuentra en la esquina superior derecha del marco . Cuando se hace clic, el editor entrará en modo de vista previa, los paneles inferiores de izquierda y derecha se ocultarán y no se modificarán las inserciones en el marco, todas las inserciones se presentarán delante de usted en 1:1.

> ### Navegadores soportados
>
> Le recomendamos que abra ILLA Builder en Google Chrome.

### Editor de acciones

El editor de acciones se encuentra debajo del marco y es el área donde escribe código para interactuar con los datos de los recursos. En esta área puede encontrar operaciones creadas, seleccionar el recurso operativo actual, escribir sentencias de consultas y configurar transformadores. o puede editar la consulta en el editor de acciones y al actualizar la consulta el editor de APP actualizará automáticamente los resultados de su acción.

Al introducir sentencias de acción SQL, puede elegir usar el modo SQL para editarlas a través de código plano, o puede elegir el modo GUI para modificarlos bajo el framework de acción que le damos para completar rápidamente la edición de la acción. o tutoriales detallados, por favor vea la documentación operativa.

### Lista de acciones

La lista de acciones se encuentra en el lado izquierdo del panel de configuración, donde puedes encontrar la acción que necesitas para usarla y modificarla en el editor de ediciones. o también puede añadir una nueva acción aquí, ver detalles del proceso de creación.

### Espacio de trabajo de datos

El espacio de trabajo de datos se encuentra en el panel a la izquierda del marco, donde puede ver las inserciones, y el estado global del editor, todos mostrados en formato Json. También puede editar los datos seleccionándolos en el área de trabajo o eliminándolos haciendo clic en el botón Más configuraciones.

### Insert

El Encarte se encuentra en el panel a la derecha del fotograma, y puede mover la inserción al fotograma arrastrándola y soltándola con el ratón. Puede encontrar el inserto que necesita por tipo de inserción, o puede buscarlo y encontrarlo rápidamente.

[//]: # "[Image]"

### Inspeccionar

Inspect se encuentra en el panel a la derecha del marco. Cuando seleccione una inserción en el marco o espacio de trabajo de datos, el panel a la derecha cambiará automáticamente a la página de inserción. Desde esta página puede establecer variables, actualizar el nombre de inserción, crear gestores de eventos o realizar una operación de eliminación de inserciones. La barra de herramientas del inserto también se establecerá de forma diferente para diferentes insertos, de modo que pueda cambiar el contenido de las diferentes inserciones.
