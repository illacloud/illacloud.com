<Heading hidden>Generación SQL</Heading>

Apoyamos la generación de SQL en estos recursos: PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB y ClickHouse. Puede ingresar el lenguaje natural y obtener una sentencia SQL lista para ejecutar cuando utilice estos recursos. 

**Introduce la página de edición de la aplicación y crea una acción usando uno de los recursos anteriores.**
